  • Random Famous Quotes API up again

    published 2 years ago

    First of all, I apologize for the problems lately with the API, I am using this API myself in a couple of projects with my clients, so I understand the frustration when this was not working. I have recently switched servers and I don't plan on switching again, so the API will remain up from now on.

    If you have more categories and quotes you want to contribute with, just open a ticket to get in touch with me and we can see how you can send me either a sql import file with the data or just a CSV or Excel file, what I need is these fields: quote, author, category.

    If anyone is having problems making the API work for the different categories - currently only "famous" and "movies" - please take a look at this sample cURL call I do to make it work, and adjust to your needs: 

                $ch = curl_init();
            curl_setopt( $ch, CURLOPT_URL, 'https://andruxnet-random-famous-quotes.p.mashape.com/' );
            curl_setopt( $ch, CURLOPT_RETURNTRANSFER, 1 );
            curl_setopt( $ch, CURLOPT_POSTFIELDS, 'cat=' . $category_name );
            curl_setopt( $ch, CURLOPT_HTTPHEADER, array( 'X-Mashape-Authorization: $mashape_key' ) );

            $quote_data = json_decode( curl_exec( $ch ) );
            curl_close( $ch );

    Thanks for hanging in there using the API.
