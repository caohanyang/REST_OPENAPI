Output Formats

All Eventful API methods are available for use via the REST XML, JSON, or YAML interfaces. To switch from one fromat to another just change /rest to /json or /yaml depending on your need.

For example this XML REST call
http://api.eventful.com/ rest /events/search?...&location=San+Diego
would be converted to
http://api.eventful.com/ json /events/search?...&location=San+Diego
for a JSON response, or converted to this
http://api.eventful.com/ yaml /events/search?...&location=San+Diego
for a YAML response.
