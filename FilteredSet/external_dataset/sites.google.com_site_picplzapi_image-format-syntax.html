Image Format Syntax

In any API call that returns pics, you can request that it gives you links to a couple of different picture formats. There is a default set of formats, but you can also request a custom list. However, just because you request a specific size doesn't mean you are going to get it. We have a number of predefined sizes, and we will return the one closest to the size requested. There are also a few methods of working with the differences in screen pixel density between devices.Â
