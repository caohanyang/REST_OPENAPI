Get a Venue's Links

Description

Returns URLs or identifiers from third parties that have been applied to this venue.

Request

GET https://api.foursquare.com/v2/venues/VENUE_ID/links

Authentication

User or userless authentication.

Parameters

Name Example Description
VENUE_ID AVNU234 required The venue ID to retrieve links for.

Response Fields

Field Description
provider For now just contains an id of a provider, e.g. nymag .
url If present, a URL for additional information about this venue from this provider.
linkedId If present, the identifer used by this provider to identify this venue.
