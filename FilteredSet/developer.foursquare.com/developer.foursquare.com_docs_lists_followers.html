List Followers

https://api.foursquare.com/v2/ lists/LIST_ID/followers
Returns a count and items of users following this list.

Note: Only valid on user-created lists
HTTP Method GET
Requires Acting User No ( learn more )
Modes supported foursquare ( learn more )
Parameters
All parameters are optional, unless otherwise indicated.
LIST_ID HZXXY3Y required id for a user-created list
Response fields
followers Returns a pageable list of compact user .
