Like a Check-in

Description

Allows the acting user to like or unlike a checkin.

Request

POST https://api.foursquare.com/v2/checkins/CHECKIN_ID/like

Authentication

User authentication.

Parameters

Name Example Description
CHECKIN_ID XXX123YYYY required The ID of the check-in to like or unlike.
set 1 If 1 , like this checkin. If 0 unlike (un-do a previous like) it. Default value is 1 .

Response Fields

Field Description
likes A count and groups of users who like this checkin. Groups generally include friends and others (indicating the relationship to the acting user), but are subject to change.
