Get Lists a Venue is On

Description

Returns the lists that this venue appears on.

Request

GET https://api.foursquare.com/v2/venues/VENUE_ID/listed

Authentication

User or userless authentication.

Parameters

Name Type Description
VENUE_ID HZXXY3Y Identity of a venue to get lists for.
group edited can be created , edited , followed , friends , other . If no acting user is present, only other is supported.
limit 30 Number of results to return, up to 200.
offset 300 Used to page through results. Must specify a group

Response Fields

Field Description
id A unique identifier for this list.
name The user-entered name for this list.
description The user-entered list description.
user The user who created this list.
following Boolean indicating whether the acting user is following this list. Absent if there is no acting user.
followers count and items of users who follow this list. All items may not be present.
editable Boolean indicating whether the acting user can edit this list. Absent if there is no acting user.
collaborative Boolean indicating whether this list is editable by the owner’s friends.
collaborators count and items of users who have edited this list. All items may not be present. ✓
canonicalUrl The canonical URL for this list, e.g. https://foursquare.com/dens/list/a-brief-history-of-foursquare
photo A photo for this list.
venueCount The number of unique venues on the list.
visitedCount The number of venues on the list visited by the acting user. Absent if there is no acting user.
listItems count of entries on this list.
