Tag Endpoints

  • GET /tags/ tag-name Get information about a tag object.
  • GET /tags/ tag-name /media/recent Get a list of recently tagged media.
  • GET /tags/search Search for tags by name.

GET /tags/ tag-name

  • Response
https://api.instagram.com/v1/tags/{tag-name}?access_token=ACCESS-TOKEN 
{
    "data": {
        "media_count": 472,
        "name": "nofilter",
    }
}

Get information about a tag object.

Requirements
Scope: public_content
Parameters
access_token A valid access token.

Get a list of recently tagged media.

Requirements
Scope: public_content
Parameters
count Count of tagged media to return.
access_token A valid access token.
max_tag_id Return media after this max_tag_id.
min_tag_id Return media before this min_tag_id.

Search for tags by name.

Requirements
Scope: public_content
Parameters
q A valid tag name without a leading #. (eg. snowy, nofilter)
access_token A valid access token.
