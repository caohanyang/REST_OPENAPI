An example application demonstrating the capabilities of the Penn BioTagger from the University of Pennsylvania. The application includes

  • A tokeniser tuned for biomedical text
  • A series of taggers, based on the MALLET machine learning library, to recognise various biomedical entity types
    • gene
    • genomic variation
    • malignancy type

Default annotations
:gene Expressions denoting genes
:malignancy-type Expressions denoting malignancy types
:location Expressions relating to genomic variation
:state-original
:state-altered
:variation
:type
Additional annotations available if selected
:Token The individual tokens of the text
