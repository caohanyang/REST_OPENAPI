Tokeniser, sentence splitter and part-of-speech tagging using the Hunpos HMM-based tagger.

This application tags Hungarian language text using the hu_szeged_kr model from the Hunpos distribution.

Default annotations
:Token The individual tokens of the text, with "category" feature for POS
Additional annotations available if selected
:SpaceToken The spaces between tokens
:Sentence Sentences detected by the sentence splitter
