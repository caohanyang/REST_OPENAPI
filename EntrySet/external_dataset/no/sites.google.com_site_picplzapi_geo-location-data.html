Geolocation Data

Geolocation data is attached to pics. If a pic has Geolocation data it will be under the attribute location. There will be a lat, and lon. Not all pictures have geolocation data. Any API call that returns a set of pics can include Geolocation data, but not by default. To have an API call return Geolocation data you must pass the query parameter include_geo=1.
