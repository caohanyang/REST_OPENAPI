Wrappers

If you're working with our API, the first thing you should do is grab the wrapper of your choice below. They cover the entire API and make it a piece of cake to get started.

Ruby

A Ruby gem which implements the complete functionality of the API and makes it easy to get started in Ruby.

Visit the createsend-ruby page

Python

A Python module which implements the complete functionality of the API and makes it easy to get started in Python.

Visit the createsend-python page

PHP

A PHP library which implements the complete functionality of the API and makes it easy to get started in PHP.

Visit the createsend-php page

.NET

A .NET assembly which implements the complete functionality of the API and makes it easy to get started in .NET.

Visit the createsend-dotnet page

Java

A Java library which implements the complete functionality of the API and makes it easy to get started in Java.

Visit the createsend-java page

Objective-C

An Objective-C library for talking to the Campaign Monitor API from Cocoa & Cocoa Touch applications.

Visit the createsend-objectivec page

Community supported wrappers

While not supported by Campaign Monitor, these wrappers are maintained by an active community of developers well versed in the language the wrapper is developed for.

Perl

A Perl module originally written by Jeffery Candiloro.

Visit the Net::CampaignMonitor module

Unofficial wrappers

The following wrappers have been contributed by our amazing customers. If you're interested in contributing a library or wrapper for your language of choice, we'd love to list it for you here. When using these wrappers please be aware that we are unable to offer support, so tread carefully!

ColdFusion

A ColdFusion wrapper in the form of ColdFusion CFCs which implements the complete Campaign Monitor API.

Visit the ColdFusion wrapper page

CakePHP

This CakePHP wrapper lets you easily manage clients, campaigns, lists and subscribers in your Campaign Monitor account.

Visit the CakePHP wrapper page

NodeJS

A NodeJS wrapper to create and send the campaigns.

Visit the NodeJS wrapper page
