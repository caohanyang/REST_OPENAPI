Amount

A complex type that describes an amount value as represented by a currency.

Types that use Amount:

    Deposit , SellingLimit , ShippingOption , ShippingService

Calls that use Amount:

    /fulfillment_policy POST , /fulfillment_policy/get_by_policy_name GET , /fulfillment_policy/{fulfillmentPolicyId} GET , /payment_policy POST , /payment_policy/get_by_policy_name GET , /payment_policy/{payment_policy_id} PUT , /payment_policy/{paymentPolicyId} GET , /privilege GET

Fields

currency ( CurrencyCodeEnum ) [0..?]
The currency in which the amount value is expressed. The currency is represented as a 3-letter ISO4217 currency code.
See the Field Index to learn which calls use currency.

value ( string ) [0..?]
The value of the amount in the specified currency.
See the Field Index to learn which calls use value.
