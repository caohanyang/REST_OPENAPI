SalesTaxBase

A container that describes the how the sales tax rate is calculated.

Call that uses SalesTaxBase:

    /sales_tax/{countryCode}/{jurisdictionId} PUT

Fields

salesTaxPercentage ( string ) [0..?]
The sales tax rate, as a percentage of the sale.
See the Field Index to learn which calls use salesTaxPercentage.

shippingAndHandlingTaxed ( boolean ) [0..?]
Set this value to true to indicate that shipping and handling charges are taxed.
See the Field Index to learn which calls use shippingAndHandlingTaxed.
