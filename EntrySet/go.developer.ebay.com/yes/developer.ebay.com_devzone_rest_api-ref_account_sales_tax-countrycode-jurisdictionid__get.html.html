Get a Sales Tax Table Entry

GET /sales_tax/{countryCode}/{jurisdictionId}

This call gets the current tax table entry for a specific tax jurisdiction. Specify the jurisdiction to retrieve using the countryCode and jurisdictionId path parameters.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/account/v1/sales_tax/{countryCode}/{jurisdictionId}

URI parameters

Parameter Type Required? Meaning
countryCode string Required A two-letter ISO 3166-1 Alpha-2 .

Applicable values are from CountryCodeEnum .
jurisdictionId string Required Unique ID for a sales tax jurisdiction.

All requests made to eBay REST operations require you to provide the authorization HTTP header for authentication.

Response field descriptions



Output Container/Field Type Occurrence Meaning
countryCode string Conditionally The country code identifying the country to which this tax rate applies.

Applicable values are from CountryCodeEnum :See countryCode .
Code so that your app gracefully handles any future changes to this list.
salesTaxJurisdictionId string Conditionally A unique ID that identifies the sales tax jurisdiction to which the tax rate applies (for example a state within the United States).
salesTaxPercentage string Conditionally The sales tax rate (as a percentage of the sale) applied to sales transactions made in this country and sales tax jurisdiction.
shippingAndHandlingTaxed boolean Conditionally Set this value to true to indicate that shipping and handling charges are taxed.

Sample: Get a sales tax table entry

This call gets a specific sales tax table entry.

Description

This call gets the sales tax table entry specified by the path parameters countryCode and jurisdictionId .

Input

This call does not use a request payload; supply the county and tax jurisdiction of the tax entry you want to get in the path of the operation URI. 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/account/v1/sales_tax/US/NY

Output

If the call is successful, it returns an HTTP status code of "200 OK" and the tax table entry that was configured for the specified sales tax jurisdiction. 

JSON format.
{
  "salesTaxPercentage": "4.0",
  "shippingAndHandlingTaxed": true,
  "countryCode": "US",
  "salesTaxJurisdictionId": "NY"
}
