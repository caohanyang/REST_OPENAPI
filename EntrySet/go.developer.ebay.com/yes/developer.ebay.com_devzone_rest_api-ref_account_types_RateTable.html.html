RateTable

A complex type that contains information pertaining to a shipping-rate table.

Type that uses RateTable:

    RateTableResponse

Call that uses RateTable:

    /rate_table GET

Fields

countryCode ( CountryCodeEnum ) [0..?]
A two-letter ISO 3166-1 Alpha-2 code representing the source or listing country.
See the Field Index to learn which calls use countryCode.

locality ( ShippingOptionTypeEnum ) [0..?]
The region covered by the shipping rate table, either DOMESTIC or INTERNATIONAL.

DOMESTIC indicates that the shipping-rate table applies to regions within country from where the item ships (the source country) while INTERNATIONAL indicates that the shipping-rate table applies to regions outside of the country from where an item is shipped.
See the Field Index to learn which calls use locality.

name ( string ) [0..?]
The user-defined name for the shipping-rate table. Sellers can access Seller Hub to assign names to their shipping-rate tables.
See the Field Index to learn which calls use name.

rateTableId ( string ) [0..?]
A unique eBay-assigned ID for the shipping-rate table.

Use this ID value in the RateTableDetails container of the Add/Revise/Relist calls in the Trading API. If locality for a rate table is set to DOMESTIC , pass the ID value in the RateTableDetails.DomesticRateTableId field. Otherwise, if locality is INTERNATIONAL , pass the ID value in RateTableDetails.InternationalRateTableId .
See the Field Index to learn which calls use rateTableId.
