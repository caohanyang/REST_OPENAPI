Delete a Return Policy

DELETE /return_policy/{return_policy_id}

This call deletes a return policy. Supply the ID of the policy you want to delete in the returnPolicyId path parameter. Note that you cannot delete the default return policy.

Resource URI (production) 

DELETE https://api.ebay.com/sell/account/v1/return_policy/{return_policy_id}

URI parameters

Parameter Type Required? Meaning
return_policy_id string Required A unique eBay-assigned ID for a return policy.
