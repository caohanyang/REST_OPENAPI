ExternalPaymentMethodTypeEnum

An enum value defining the supported payment methods.

No calls use ExternalPaymentMethodTypeEnum.

Enumeration Values

Value Description
BANK_CASH_TRANSFER Payment via cash to or from a bank.
CASH Payment via a cash transaction.
CHECK Payment via a check.
CREDIT_CARD Payment via a credit card.
DEBIT_CARD Payment via a debit card.
OTHER Payment via another method.
PAYPAL Payment via a PayPal.
* See the Enumeration Index to see exact use of each enumeration value in the API.
