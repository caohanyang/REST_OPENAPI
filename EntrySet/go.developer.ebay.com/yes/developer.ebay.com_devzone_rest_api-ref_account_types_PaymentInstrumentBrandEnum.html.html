PaymentInstrumentBrandEnum

An enum type defining the supported credit card brand names.

Type that uses PaymentInstrumentBrandEnum:

    PaymentMethod

Calls that use one or more values of PaymentInstrumentBrandEnum:

    /payment_policy POST , /payment_policy/get_by_policy_name GET , /payment_policy/{payment_policy_id} PUT , /payment_policy/{paymentPolicyId} GET

Enumeration Values

Value Description
AMERICAN_EXPRESS Payment instrument brand name for American Express.
DISCOVER Payment instrument brand name for Discover.
MASTERCARD Payment instrument brand name for MasterCard.
VISA Payment instrument brand name for Visa.
