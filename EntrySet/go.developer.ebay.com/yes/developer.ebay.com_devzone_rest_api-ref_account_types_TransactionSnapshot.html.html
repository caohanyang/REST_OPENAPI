TransactionSnapshot

A snapshot of a merchant's transactions, based on a specified lookback period.

No calls use TransactionSnapshot.

Fields

calendarWindowAggregatedMetrics ( SoldAggregatedMetrics )
lookbackPeriod ( LookbackPeriod )
rollingWindowAggregatedMetrics ( SoldAggregatedMetrics )
soldInventoryValue ( Amount )
soldQuantity ( integer )


calendarWindowAggregatedMetrics ( SoldAggregatedMetrics ) [0..?]
Not used by any call.

A list of the aggregated metrics for sold items during the specified lookback period.
See the Field Index to learn which calls use calendarWindowAggregatedMetrics.

lookbackPeriod ( LookbackPeriod ) [0..?]
Not used by any call.

The duration of time, in the past, to which the transaction snapshot (history) applies.
See the Field Index to learn which calls use lookbackPeriod.

rollingWindowAggregatedMetrics ( SoldAggregatedMetrics ) [0..?]
Not used by any call.

A list of the aggregated metrics for sold items during the rolling window during the specified lookback period.
See the Field Index to learn which calls use rollingWindowAggregatedMetrics.

soldInventoryValue ( Amount ) [0..?]
Not used by any call.

The total inventory value that was sold during the specified lookback period.
See the Field Index to learn which calls use soldInventoryValue.

soldQuantity ( integer ) [0..?]
Not used by any call.

The total quantities sold in during the specified lookback period.
See the Field Index to learn which calls use soldQuantity.
