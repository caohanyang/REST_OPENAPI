Call Index

This API Reference describes the elements and attributes for each call in the Account API.

The Account API gives sellers the ability to configure their eBay seller accounts, including the seller's policies (the Fulfillment Policy, Payment Policy, and Return Policy) and other preferences.

Call Summary Samples
FULFILLMENT_POLICY
Create a Fulfillment Policy
POST /fulfillment_policy 		Creates a fulfillment policy for the associated seller account. view
Delete a Fulfillment Policy
DELETE /fulfillment_policy/{fulfillmentPolicyId} 		Deletes a fulfillment policy from the associated seller account. view
Get Fulfillment Policies by Marketplace
GET /fulfillment_policy 		Retrieves all the fulfillment policies associated with the specified marketplace. view
Get a Fulfillment Policy by ID
GET /fulfillment_policy/{fulfillmentPolicyId} 		Retrieves the fulfillment policy associated with the specified fulfillment policy ID. view
Get a Fulfillment Policy by Name
GET /fulfillment_policy/get_by_policy_name 		Retrieves the fulfillment policy associated with the specified fulfillment policy name. view
Update a Fulfillment Policy
PUT /fulfillment_policy/{fulfillmentPolicyId} 		Updates the fulfillment policy associated with the specified fulfillment policy ID. view
