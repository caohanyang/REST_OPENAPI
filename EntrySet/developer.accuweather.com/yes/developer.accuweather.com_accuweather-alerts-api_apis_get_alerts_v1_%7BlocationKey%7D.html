Alerts by LocationKey

Returns alert data for a specific location. Alert searches require a location key. Please use the Locations API to obtain the location key for your desired location. By default, a truncated version of the alert data is returned. The full object can be obtained by passing "details=true" into the url string.

Resource URL

http://dataservice.accuweather.com /alerts/v1/{locationKey}

Query Parameters

Name Values Description
apikey
(required)

Provided API Key

language

String indicating the language in which to return the resource

details

Boolean value specifies whether or not to include full details in the response.

Request Body

Response Parameters

Parameter Type Description
CountryCode string Unique country code ID.
AlertID int64 Unique numeric id that identifies an alert for a particular country.
Description.Localized string Description of the alert in the local dialect set with language code in URL. Default is US English (en-us).
Description.English string Description of the alert in US English.
Category string Category of the alert.
Priority int32 Number signifying the importance or ranking order of the given alert within the country in which it has originated. A lower number signifies a higher priority. For example, 1 is the highest priority.
Type string Abbreviated form of the alert description.
TypeId string ID associated with the alert description.
Class string Classification of the alert. Not available for all countries.
Level string Severity identifier for the alert. Not available for all countries.
Color.Name string Descriptive name of the color.
Color.Red int32 The red RGB component of the color.
Color.Green int32 The green RGB component of the color.
Color.Blue int32 The blue RGB component of the color.
Color.Hex string The hexadecimal number that describes the color.
Source string Provider of the alert data.
SourceId int32 ID associated with the provider of the alert data.
Disclaimer string Disclaimer that must be displayed with the alert data.
Area.Name string Name of the area that encompasses the location that was requested and falls under the current alert.
Area.StartTime string Start time of the alert, displayed in ISO8601 format.
Area.EpochStartTime int64 Start time of the alert, displayed as the number of seconds that have elapsed since January 1, 1970 (midnight UTC/GMT). May be NULL.
Area.EndTime string End time of the alert, displayed in ISO8601 format.
Area.EpochEndTime int64 End time of the alert, displayed as the number of seconds that have elapsed since January 1, 1970 (midnight UTC/GMT). May be NULL.
Area.LastAction.Localized string Status of the alert in the local dialect set with language code in URL. Default is US English (en-us).
Area.LastAction.English string Status of the alert in US English.
Area.Text string Full text associated with the alert.
Area.LanguageCode string Language of the alert text.
Area.Summary string Alert summary, in English.
MobileLink string Link to severe weather alerts for the requested location on AccuWeather`s mobile site.
Link string Link to severe weather alerts for the requested location on AccuWeather`s web site.
