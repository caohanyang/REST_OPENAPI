API Method /performers/get

Given a performer ID, returns the performer data associated with that performer.

Arguments

authentication
This method controls access via authentication .
id string
The ID of the requested performer.
show_events boolean
If true, include the events list. (see below)
image_sizes string
A comma separted list of image_sizes desired. See Faq for list of possible image sizes. (optional)

Example Request

Here's an example request URL:

http://api.eventful.com/rest/performers/get? ... &id=P0-001-000000045-2

Output Parameters

Output from this function is an XML block ("performer") with the following constructs.
id string
The unique ID for this performer.
url string
The page URL for this performer on eventful.com.
name string
The name of this performer.
is_human boolean
True if this performer is human (a singer, a band, a company) rather than abstract (a film, an idea, a musical composition)
short_bio string
A short biography of this performer.
long_bio string
A detailed biography of this performer.
demand_count integer
The number of demands for this performer.
demand_member_count integer
The number of people who joined demands for this performer.
event_count integer
The number of events involving this performer.
created datetime
The date & time the performer was initially created. (e.g. '2006-03-17 16:55:00')
creator string
The username of the performer's creator.
withdrawn boolean
True if this performer has been marked as withdrawn.
withdrawn_note string
If this performer has been marked as withdrawn, a short reason why.
links array
An array of link entries
comments array
An array of comment entries
images array
An array of image entries
tags array
An array of tag entries
demands array
An array of demand entries
events array
An array of event entries. This list is only included if the show_events parameter is true.

Error Codes

This method may return these errors in an error document :

Authentication error
The specified user was not found or provided invalid credentials.
Missing parameter
A required parameter was not provided.
Not found
There is no performer with that identifier.
