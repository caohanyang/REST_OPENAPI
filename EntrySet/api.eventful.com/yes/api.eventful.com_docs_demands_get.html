API Method /demands/get

Given a demand ID, returns the demand data associated with that demand.

Arguments

authentication
This method controls access via authentication .
id string
The ID of the requested demand.

Example Request

Here's an example request URL:

http://api.eventful.com/rest/demands/get? ... &id=D0-001-000000045-2

Output Parameters

Output from this function is an XML block ("demand") with the following constructs.
id string
The unique ID for this demand.
description string
The demand description.
status string
The current status of the demand. (e.g. 'demanded', 'criticalmass')
location string
member_count integer
The number of members who have joined this demand to date.
created datetime
The date & time the demand was initially created. (e.g. '2006-03-17 16:55:00')
creator string
The username of the demand's creator.
withdrawn boolean
True if this demand has been marked as withdrawn.
withdrawn_note string
If this demand has been marked as withdrawn, a short reason why.
performer struct
A performer entry with the following structure:
id string
The performer ID for this demand.
name string
The name of this performer.
is_human boolean
True if this performer is human (a singer, a band, a company) rather than abstract (a film, an idea, a musical composition)
short_bio string
A short biography of this performer.
long_bio string
A detailed biography of this performer.
created datetime
The date & time the performer was initially created. (e.g. '2006-03-17 16:55:00')
creator string
The username of the performer's creator.
withdrawn boolean
True if this performer has been marked as withdrawn.
withdrawn_note string
If this performer has been marked as withdrawn, a short reason why.
events array
An array of event entries
links array
An array of link entries
comments array
An array of comment entries
images array
An array of image entries
tags array
An array of tag entries

Error Codes

This method may return these errors in an error document :

Authentication error
The specified user was not found or provided invalid credentials.
Missing parameter
A required parameter was not provided.
Not found
There is no demand with that identifier.
