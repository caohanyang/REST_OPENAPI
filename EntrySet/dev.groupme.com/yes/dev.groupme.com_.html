API Documentation

Welcome to the API documentation site. Here you can find everything you need to learn how to develop apps using our API. Our API will enable you to enhance existing apps with our group messaging abilities, build interesting new experiences, or simply add a little spice to your existing groups.

Questions about our API? We'd love to hear them. Check out our GroupMe API Support Google Group or simply email groupme-api-support@googlegroups.com . By using the GroupMe API you accept the GroupMe API License Agreement .

Getting Started

To get started, log in to the right in order to create apps for yourself.

Once registered, browse our REST API reference to see what kinds of actions are available.

We also have specific tutorials on authenticating with GroupMe , using our real-time message system and bots, which let programs inject messages into your groups as well as listen to messages published in them.
