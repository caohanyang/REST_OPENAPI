Client Libraries

Client libraries to simplify access to the GeoNames web services exist in a couple of programming languages.
  • Java Client developed by the GeoNames team.
  • cloudrail Universal API.
  • Objective-C
  • GeoNames Ruby (fork from Ruby Client )
  • GeoNames API Ruby
  • Drupal : Drupal Module and Project Page
  • geopy - Geocoding Toolbox for Python
  • Perl GeoNames Client
  • Lisp Client
  • Elixir Client
  • Pear PHP
  • Leaflet.Geonames
  • Zend Framework PHP
  • SOLMETRA Maps (PHP)
  • Yii (PHP)
  • PHP Laravel
  • R-Project
  • .NET NGeo sourcecode and NuGet Package
  • .NET WCF
  • .NET 3.5
  • GeoExt
  • GeoNames api for actionscript 3
  • jeoQuery - jQuery for GeoNames
  • Geode for NodeJS

  • NodeJs geonames.js library

Plugins

  • opensearch plugin for Firefox and IE7
  • Mozilla Plugin by Emilio Gomez Fernandez
  • Geocode plugin for Geomajas
  • Bubble.is plugin

Frameworks

  • Gisgraphy : an open source framework for GeoNames data, with importer, geolocalisatoin and fulltext search.
  • NGeoNames :.Net package. provides classes for downloading and parsing files from GeoNames.org and provides (reverse) geocoding methods like NearestNeighbourSearch() and RadialSearch() on the downloaded dataset(s).
