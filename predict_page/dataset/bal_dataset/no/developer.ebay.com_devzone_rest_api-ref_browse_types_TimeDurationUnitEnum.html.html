TimeDurationUnitEnum

Enumeration type that defines the values that indicate period of time being used.

Type that uses TimeDurationUnitEnum:

    TimeDuration

Call that uses one or more values of TimeDurationUnitEnum:

    /item/{item_id} GET

Enumeration Values

Value Description
BUSINESS_DAY Indicates the time duration is based on the number of business days, meaning 'working days'. So it excludes all holidays for the location and in some locations can include Saturday.
CALENDAR_DAY Indicates the time duration is based on the calendar, meaning Sunday through Saturday and does not exclude holidays.
DAY Indicates the time duration is based on the number of days.
HOUR Indicates the time duration is based on the number of hours.
MILLISECOND Indicates the time duration is based on the number of milliseconds.
MINUTE Indicates the time duration is based on the number of minutes.
MONTH Indicates the time duration is based on the number of months.
SECOND Indicates the time duration is based on the number of seconds.
YEAR Indicates the time duration is based on the number of years.
* See the Enumeration Index to see exact use of each enumeration value in the API.
