Amount

This type defines the information for a monetary value and the conversion of the value into another currency.

No calls use Amount.

Fields

currency ( CurrencyCodeEnum ) [0..?]
Not used by any call.

The currency involved in a monetary transaction. Generally, this is the currency used by the country of eBay site offering the item.
See the Field Index to learn which calls use currency.

value ( string ) [0..?]
Not used by any call.

The dollar value of the currency specified in the currency field. The value of currency defaults to the standard currency used by the country of the eBay site offering the item.
See the Field Index to learn which calls use value.
