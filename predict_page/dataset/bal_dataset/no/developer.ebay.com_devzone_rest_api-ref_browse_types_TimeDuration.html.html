TimeDuration

An amount of time.

Type that uses TimeDuration:

    ItemReturnTerms

Call that uses TimeDuration:

    /item/{item_id} GET

Fields

unit ( TimeDurationUnitEnum ) [0..?]
Indicates the period of time being used.
See the Field Index to learn which calls use unit.

value ( integer ) [0..?]
Indicates the number of years, months, days, etc. in the defined period. If value is 30 , and unit is DAY , the defined period is 30 days.
See the Field Index to learn which calls use value.
