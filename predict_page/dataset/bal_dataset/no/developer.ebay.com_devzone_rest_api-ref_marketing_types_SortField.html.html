SortField

This type defines the fields that control how items in result sets are sorted.

Call that uses SortField:

    /promotion GET

Fields

ascending ( boolean ) [0..?]
Indicates if the results are sorted in ascending order.
See the Field Index to learn which calls use ascending.

field ( string ) [0..?]
Specifies the name of a field in which to sort promotions using the value of that field.
See the Field Index to learn which calls use field.
