CancelStateEnum

This enumerated type contains a list of the states that can apply to an order with regard to cancellation.

Type that uses CancelStateEnum:

    CancelStatus

Calls that use one or more values of CancelStateEnum:

    /order GET , /order/{orderId} GET

Enumeration Values

Value Description
CANCELED The order has been cancelled.
IN_PROGRESS One or more cancellation requests have been made against the order.
NONE_REQUESTED No cancellation requests have been made against the order.
