Ad

Type the defines the fields for an ad.

Types that use Ad:

    AdPagedCollection , Ads

Calls that use Ad:

    /ad_campaign/{campaign_id}/ad GET , /ad_campaign/{campaign_id}/ad/{ad_id} GET , /ad_campaign/{campaign_id}/get_ads_by_inventory_reference GET

Fields

adId ( string )
bidPercentage ( string )
inventoryReferenceId ( string )
inventoryReferenceType ( InventoryReferenceTypeEnum )
listingId ( string )


adId ( string ) [0..?]
Identifier of an ad. This ID was generated when the ad was created.
See the Field Index to learn which calls use adId.

bidPercentage ( string ) [0..?]
The bid percentage associated with the listing. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20


See the Field Index to learn which calls use bidPercentage.

inventoryReferenceId ( string ) [0..?]
The seller's inventory reference ID for a listing. An inventory reference is either the ID of a listing or the ID of the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .
See the Field Index to learn which calls use inventoryReferenceId.

inventoryReferenceType ( InventoryReferenceTypeEnum ) [0..?]
Indicates the inventory reference ID type of the inventoryReferenceId . You must always pass in both inventory_reference_id and inventory_reference_type .
See the Field Index to learn which calls use inventoryReferenceType.

listingId ( string ) [0..?]
The eBay listing identifier. This is generated when the listing was created.
See the Field Index to learn which calls use listingId.
