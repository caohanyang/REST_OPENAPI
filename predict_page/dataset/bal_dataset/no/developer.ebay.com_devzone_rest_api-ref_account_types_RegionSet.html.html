RegionSet

A list of regions (defined by a regionType and a region name), including region names that are specifically included and excluded from a regionType .

Types that use RegionSet:

    FulfillmentPolicy , FulfillmentPolicyRequest , SetFulfillmentPolicyResponse , ShippingService

Calls that use RegionSet:

    /fulfillment_policy POST , /fulfillment_policy/get_by_policy_name GET , /fulfillment_policy/{fulfillmentPolicyId} GET

Fields

regionExcluded ( array of Region ) [0..?]
A list of regions that are specifically excluded from a region set. You can use this to exclude a specific regionName from a regionType that you specifically include.
See the Field Index to learn which calls use regionExcluded.

regionIncluded ( array of Region ) [0..?]
A list of regions that are specifically included in a region set. Use this field to include a specific regionName from a regionType that you've excluded.
See the Field Index to learn which calls use regionIncluded.
