Promotion

Container for the item promotions details.

Type that uses Promotion:

    LineItem

Calls that use Promotion:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

discount ( Amount ) [0..?]
The dollar value of the promotional discount. Note: eBay Bucks and coupons are not supported.
See the Field Index to learn which calls use discount.

message ( string ) [0..?]
The text for the promotion title which is displayed on the page. For example, Buy 1 Get 1.
See the Field Index to learn which calls use message.

promotionCode ( string ) [0..?]
An identifier of the promotion that was generated by the system when the promotion was created.
See the Field Index to learn which calls use promotionCode.

promotionType ( string ) [0..?]
The value of the enumeration used when the promotions was created, which indicates the promotion type. For example, SellerDiscountedPromotionalOffer.
See the Field Index to learn which calls use promotionType.
