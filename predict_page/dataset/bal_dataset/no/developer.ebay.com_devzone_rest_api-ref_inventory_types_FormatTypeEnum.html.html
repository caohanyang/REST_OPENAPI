FormatTypeEnum

This enumeration type is used to indicate the listing format of the offer.

Types that use FormatTypeEnum:

    EbayOfferDetailsWithAll , EbayOfferDetailsWithKeys

Calls that use one or more values of FormatTypeEnum:

    /offer POST , /offer/{offerId} GET

Enumeration Value

Value Description
FIXED_PRICE This enumeration value indicates that the listing format of the offer is fixed price. With the initial release of the Inventory API, fixed price is the only supported listing format.
