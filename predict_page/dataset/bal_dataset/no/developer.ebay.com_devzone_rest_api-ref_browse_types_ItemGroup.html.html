ItemGroup

Type that defines the fields that describe the items in a group (items with variations).

Call that uses ItemGroup:

    /item_group/{item_group_id} GET

Fields

additionalImages ( array of Image )
categoryPath ( string )
description ( string )
generatedShortDescription ( string )
image ( Image )
itemGroupId ( string )
items ( array of ItemDigest )
reviewRating ( ReviewRating )
sellerProvidedDescription ( string )
shortDescription ( string )
title ( string )
variesByLocalizedAspects ( array of TypedNameValue )
warnings ( array of ErrorDetailV3 )


additionalImages ( array of Image ) [0..?]
An array of item image URLs that does not include the primary image for the item. For items with variations, such as color, these are often images showing the item in the various colors. The URL for the primary image is returned in the image.imageUrl field.
See the Field Index to learn which calls use additionalImages.

categoryPath ( string ) [0..?]
Text that shows the category hierarchy of the item. For example, iPads, Tablets & eBook Readers.
See the Field Index to learn which calls use categoryPath.

description ( string ) [0..?]
The full description of the item that was created by the seller. This can be plain text or rich content.
See the Field Index to learn which calls use description.

generatedShortDescription ( string ) [0..?]
Note: Do not use this field because it will be removed in the future. Use the shortDescription field instead.
See the Field Index to learn which calls use generatedShortDescription.

image ( Image ) [0..?]
The URL to the primary image of the item. The other images of the item are returned in the additionalImages field.
See the Field Index to learn which calls use image.

itemGroupId ( string ) [0..?]
The uniqie identifier for the item group. For example, 1307809542
See the Field Index to learn which calls use itemGroupId.

items ( array of ItemDigest ) [0..?]
An array of items in this group.
See the Field Index to learn which calls use items.

reviewRating ( ReviewRating ) [0..?]
The product rating details.
See the Field Index to learn which calls use reviewRating.

sellerProvidedDescription ( string ) [0..?]
Note: Do not use this field because it will be removed in the future. Use the description field instead.
See the Field Index to learn which calls use sellerProvidedDescription.

shortDescription ( string ) [0..?]
A snippet of the full description of the item in plain text.
See the Field Index to learn which calls use shortDescription.

title ( string ) [0..?]
The seller created title of the item.
See the Field Index to learn which calls use title.

variesByLocalizedAspects ( array of TypedNameValue ) [0..?]
A list of type/name/value data that describe the variations of item. The variation can be a single value or a list of values.

For example: 
{
   "type": "STRING",
    "name": "Style",
    "value": "Blouse"
 },





See the Field Index to learn which calls use variesByLocalizedAspects.

warnings ( array of ErrorDetailV3 ) [0..?]
An array of warning messages. These type of errors do not prevent the call from executing but should be checked.
See the Field Index to learn which calls use warnings.
