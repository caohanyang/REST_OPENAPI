BulkDeleteAdsByInventoryReferenceRequest

Type that defines the fields for the remove ads by inventory reference ID request.

    /ad_campaign/{campaign_id}/bulk_delete_ads_by_inventory_reference POST

Field

requests ( array of DeleteAdsByInventoryReferenceRequest ) [0..?]
An array of the seller's inventory references of the ads to remove.
