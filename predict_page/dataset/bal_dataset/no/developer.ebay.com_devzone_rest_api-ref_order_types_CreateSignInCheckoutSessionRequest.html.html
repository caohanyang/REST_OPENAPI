CreateSignInCheckoutSessionRequest

Type that defines the fields for creating a eBay member checkout session.

Call that uses CreateSignInCheckoutSessionRequest:

    /checkout_session/initiate POST

Fields

creditCard ( CreditCard ) [0..?]
Container for the buyer's credit card information.
See the Field Index to learn which calls use creditCard.

lineItemInputs ( array of LineItemInput ) [0..?]
Type that defines the fields for a line item in a eBay member checkout session. The maximum number of lines item in a checkout session is 4.
See the Field Index to learn which calls use lineItemInputs.

shippingAddress ( ShippingAddress ) [0..?]
Container for the shipping address in a eBay member checkout session.
See the Field Index to learn which calls use shippingAddress.
