ReturnSnapshot

A snapshot (history) of the seller's returns. The time span of the snapshot is defined by the lookbackPeriod .

No calls use ReturnSnapshot.

Fields

lookbackPeriod ( LookbackPeriod ) [0..?]
Not used by any call.

The duration of time, in the past, to which the return snapshot (history) applies.
See the Field Index to learn which calls use lookbackPeriod.

returnedInventoryValue ( Amount ) [0..?]
Not used by any call.

The total value of returned inventory over the specified lookback period.
See the Field Index to learn which calls use returnedInventoryValue.

returnedQuantity ( integer ) [0..?]
Not used by any call.

The total quantity of items returned over the specified lookback period.
See the Field Index to learn which calls use returnedQuantity.
