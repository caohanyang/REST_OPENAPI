TargetLocation

This type is used by the distanceFromPickupLocation container and is used to show the distance between the item location and the buyer's location.

Type that uses TargetLocation:

    ItemSummary

Call that uses TargetLocation:

    /item_summary/search GET

Fields

unitOfMeasure ( string ) [0..?]
This value shows the unit of measurement used to measure the distance between the location of the item and the buyer's location. This value is typically mi or km .
See the Field Index to learn which calls use unitOfMeasure.

value ( string ) [0..?]
This value indicates the distance (measured in the measurement unit in the unitOfMeasure field) between the item location and the buyer's location.
See the Field Index to learn which calls use value.
