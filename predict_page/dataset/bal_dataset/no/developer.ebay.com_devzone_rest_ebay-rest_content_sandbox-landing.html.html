Testing your eBay applications in the Sandbox

The eBay Sandbox is a self-contained virtual testing environment that mimics the live eBay production environment.

The Sandbox provides a shielded space where you can test your application, and watch it process mock eBay transactions generated by your requests to the eBay APIs without touching any live eBay listings, user accounts, or bank accounts . Testing in the Sandbox gives you the ability to fully test and debug your eBay API routines before you run them in eBay's live production environment.

The Sandbox environment

The Sandbox environment comprises the following:

  • The Sandbox API location : The URI for the Sandbox is: https://api.sandbox.ebay.com . This is the URL location you should target during the testing phase of your REST application development.
  • The Sandbox test site ( https://sandbox.ebay.com/ )
  • The Sandbox Accounts page (accessed through https://developer.ebay.com )

The Sandbox location

During the testing phase of your app development, make sure that all your calls target the Sandbox servers by using the Sandbox URI as the base APIÂ location for all your calls. This ensures your calls do not affect live users or their accounts.

The Sandbox test site

The Sandbox test site looks and behaves like the real eBay site, with the exception that all users of the test site are test accounts that you set up. While you test, log into the Sandbox test site with a test user account that represents either a buyer or a seller. Test buyers can browse and purchase the items that your test sellers list. You can configure your test seller accounts, and view the process flow, just as you would on the production eBay site.

The Sandbox Accounts page

Use this page on the eBay Developer site to configure your Sandbox test accounts. Configure both buyer and seller accounts, and use these accounts as the entities that partake in the mock transactions that you create and process during the testing phase of your app development. Note that you must have an eBay Developer Program account to log in to the Developer Program site, and configure test accounts.

Going live with your application

After you've completed the testing phase for your application and are ready to go live with your app, complete the Compatible Application Check .

The Compatible Application Check instructions page gives you a checklist to follow before going live with your application. Passing the check authorizes you to use the eBay Compatible Application logo .
