Specification

This type is used to specify product aspects for which variations within an inventory item group vary, and the order in which they appear in the listing. For example, t-shirts in an inventory item group may be available in multiple sizes and colors.

Type that uses Specification:

    VariesBy

Call that uses Specification:

    /inventory_item_group/{inventoryItemGroupKey} PUT

Fields

name ( string ) [0..?]
This is the name of product variation aspect. For example, the aspect names in this field might be "Size" and "Color" .
See the Field Index to learn which calls use name.

values ( array of string ) [0..?]
This is an array of values pertaining to the corresponding product variation aspect (specified in the name field). Below is a sample of how these values will appear under a specifications container: 
"specifications": [{
 "name": "Size",
 "values": ["Small",
 "Medium",
 "Large"],
 "order": 1 
 },
 { 
 "name": "Color",
 "values": ["Blue",
 "White",
 "Red"],
 "order": 2 
 }]

See the Field Index to learn which calls use values.
