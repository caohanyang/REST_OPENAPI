CycleTypeEnum

An enum that defines the allowable cycle types.

Type that uses CycleTypeEnum:

    Cycle

Calls that use one or more values of CycleTypeEnum:

    /seller_standards_profile GET , /seller_standards_profile/{program}/{cycle} GET

Enumeration Values

Value Description
CURRENT Indicates the evaluation that determines your current, effective seller level.
PROJECTED Indicates the seller level that would take effect on the next evaluation.
* See the Enumeration Index to see exact use of each enumeration value in the API.
