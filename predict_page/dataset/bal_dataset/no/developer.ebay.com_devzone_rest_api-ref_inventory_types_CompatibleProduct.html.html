CompatibleProduct

This type is used to specify/indicate the motor vehicles that are compatible with the corresponding inventory item.

Type that uses CompatibleProduct:

    Compatibility

Call that uses CompatibleProduct:

    /inventory_item/{sku}/product_compatibility PUT

Fields

notes ( string ) [0..?]
This field is optionally used by the seller to input any notes pertaining to the parts compatibility being specified. The seller might use this field to specify the placement of the part on a vehicle or other applicable information. This field will only be returned if specified by the seller.
See the Field Index to learn which calls use notes.

productFamilyProperties ( ProductFamilyProperties ) [0..?]
This container consists of an array of motor vehicles that are compatible with the motor vehicle or part or accessory specified by the sku value. These motor vehicle are identified by properties such as make, model, year, trim, and engine type.

Either the productIdentifier or the productFamilyProperties container (or both) can be used to specify one or more motor vehicles that are compatible with the inventory item. If the productIdentifier container is used, the product identifiers that are passed in that container must match vehicles in the eBay catalog in order for that vehicle to be added to the compatible vehicle list.
See the Field Index to learn which calls use productFamilyProperties.

productIdentifier ( ProductIdentifier ) [0..?]
This container is used in a Create Or Replace Product Compatibility call to identify a motor vehicle that is compatible with the inventory item. If the motor vehicle is found in the eBay catalog, the motor vehicle properties (the properties in the productFamilyProperties ) will automatically be populated for the vehicle.

Either the productIdentifier or the productFamilyProperties container (or both) can be used to specify one or more motor vehicles that are compatible with the inventory item. If the productIdentifier container is used, the product identifiers that are passed in that container must match vehicles in the eBay catalog in order for that vehicle to be added to the compatible vehicle list.
See the Field Index to learn which calls use productIdentifier.
