Bulk Update Ad Bids By Inventory Reference

POST /ad_campaign/{campaign_id}/bulk_update_ads_bid_by_inventory_reference

This call replaces the Ad bid for the list of seller inventory references in the specified campaign.

A seller's inventory reference ID is the identifier of a listing or the identifier of the parent of a listing with variations, such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .

You can have a maximum of 500 IDs per call.

Request field descriptions



Input Container/Field Type Occurrence Meaning
requests array of CreateAdsByInventoryReferenceRequest Optional An array of the seller's inventory reference/inventory type pairs and the bid percentage for each to be used to create ads.
requests. bidPercentage string Required The bid percentage associated with the seller's inventory reference. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. If bid percentage is not provided, the default bid percentage associated with the campaign is used. The higher the bid, the better the position of the listing on the page.

The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20




requests. inventoryReferenceId string Required The seller's inventory reference ID for a listing. This can be an ID of a listing or the ID of the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .

Maximum IDs per call: 500

requests
. inventoryReferenceType 		string Required Indicates the inventory reference ID type of the inventoryReferenceId . You must always pass in both inventory_reference_id and inventory_reference_type .

Applicable values are from InventoryReferenceTypeEnum :

INVENTORY_ITEM
Indicates the inventory reference ID is for a listing.
INVENTORY_ITEM_GROUP
Indicates the inventory reference ID is the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors.

Sample: Update Bid in Bulk Using Inventory Reference
