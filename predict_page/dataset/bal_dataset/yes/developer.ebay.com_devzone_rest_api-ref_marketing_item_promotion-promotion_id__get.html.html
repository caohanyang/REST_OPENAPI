Get Item Promotion

GET /item_promotion/{promotion_id}


This call returns the details of the specified promotion so you can review all the promotion parameters. You specify the promotion by passing in the promotion_id as a URI parameter.

You can use the Get Promotions call retrieve the IDs of the seller's promotions.

Response field descriptions

description string Conditionally The "tag line" for the offer type, such as "Save on designer shoes." It appears under the offer type text in the promotion tile that shows on the seller's All Offer page, and on the event page for the promotion.

Note: The offer type text is used as a teaser on the search results page and the listing page. It's not editable by the seller and is derived from the settings in the discountRules and discountSpecification fields. Example of offer type text: Extra 20% off when you buy 3+ Max length: 50

discountRules.discountBenefit
. amountOffItem 		Amount Conditionally The amount of discount that will be applied to an item when the promotion criteria is met. This would be used to create an 'Extra $5 off each item' offer. Every item the buyer purchases from the inventory set gets a $5 discount.

To create this promotion, you set the following discount details:

  • discountSpecification.forEachQuantity to 1
  • discountSpecification.numberOfDiscountedItems to 0
