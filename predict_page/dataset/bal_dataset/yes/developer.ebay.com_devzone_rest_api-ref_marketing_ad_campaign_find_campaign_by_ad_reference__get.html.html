Find Campaign By Ad Reference

GET /ad_campaign/find_campaign_by_ad_reference


This call retrieves the campaign containing the specified listing. You can pass in either an eBay listing ID or the seller's inventory reference ID and type.

eBay listing IDs are generated when the listing is created on eBay. A seller's inventory reference ID is the identifier of a listing or the identifier of the parent of a listing with variations, such as a shirt that is available in multiple sizes and colors.

In this call, you must pass in either listing_id or inventory_reference_id and inventory_reference_type .

Input

Resource URI (production) 

GET https://api.ebay.com/sell/marketing/v1/ad_campaign/find_campaign_by_ad_reference?
  inventory_reference_id=string&
  inventory_reference_type=string&
  listing_id=string

URI parameters

Parameter Type Required? Meaning
inventory_reference_id string Optional The seller's inventory reference ID of the listing to be used to find the campaign in which it is associated. You must always pass in both inventory_reference_id and inventory_reference_type .
inventory_reference_type string Optional The type of the seller's inventory reference ID, which is a listing or group of items. You must always pass in both inventory_reference_id and inventory_reference_type .
listing_id string Optional Identifier of the eBay listing associated with the ad.
