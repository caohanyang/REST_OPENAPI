Get Ads

GET /ad_campaign/{campaign_id}/ad


This call retrieves all the ads for the specified campaign. You can filter the result set by eBay listing IDs and reduce the number of items returned in a result set by using the limit and offset fields.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/marketing/v1/ad_campaign/{campaign_id}/ad?
  limit=string&
  listing_ids=string&
  offset=string

URI parameters

Parameter Type Required? Meaning
campaign_id string Required Identifier of a campaign in which the ads are associated. This ID was generated when the campaign was created.
limit string Optional Specifies the maximum number of result sets to return.

Default: 10
Maximum: 500


listing_ids string Required A comma separated list of the eBay listing IDs. The results are filtered to include only active ads for these listing. The results will not include all the listing IDs when other search conditions exclude them.
offset string Optional Specifies the number of ads from the top of the result set to ignore. For example, if you set this to 5, the first ad returned would be the 6th one in the list. This is used with the limit field to control the pagination of the output. This value is usually obtained from a previous response.

For example:
If offset is 10 and limit is 10, the call will return ads 11 thru 20 from the list of ads returned.
Default: 0

Response field descriptions



Output Container/Field Type Occurrence Meaning
ads array of Ad Conditionally An array of ads in the paginated collection.
ads. adId string Conditionally Identifier of an ad. This ID was generated when the ad was created.
ads. bidPercentage string Conditionally The bid percentage associated with the listing. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20


ads. inventoryReferenceId string Conditionally The seller's inventory reference ID for a listing. An inventory reference is either the ID of a listing or the ID of the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .
ads. inventoryReferenceType string Conditionally Indicates the inventory reference ID type of the inventoryReferenceId . You must always pass in both inventory_reference_id and inventory_reference_type .

Applicable values are from InventoryReferenceTypeEnum :

INVENTORY_ITEM
Indicates the inventory reference ID is for a listing.
INVENTORY_ITEM_GROUP
Indicates the inventory reference ID is the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors.

Code so that your app gracefully handles any future changes to this list.
ads. listingId string Conditionally The eBay listing identifier. This is generated when the listing was created.
href string Conditionally The link to the result set.
limit integer Conditionally The limit value that was used. This sets the maximum number of items in a result set.
next string Conditionally The link to the next set of results.
offset integer Conditionally The offset value that was used. This sets the starting position in the result set.
prev string Conditionally The link to the previous set of results.
total integer Conditionally The total number of result sets in the paginated collection.
