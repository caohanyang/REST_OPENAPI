Bulk Update Ad Bids By Listing ID

POST /ad_campaign/{campaign_id}/bulk_update_ads_bid_by_listing_id

This call replaces the ads for the specified campaign based on a list of eBay listing IDs.

eBay listing IDs are generated when the listing is created on eBay.

Input

See also Samples .

Resource URI (production) 

POST https://api.ebay.com/sell/marketing/v1/ad_campaign/{campaign_id}/bulk_update_ads_bid_by_listing_id

URI parameters

campaign_id string Required Identifier of a campaign in which to update the ads. This ID was generated when the campaign was created.

Request field descriptions



Input Container/Field Type Occurrence Meaning
requests array of CreateAdRequest Optional An array of the eBay listing IDs and the bid percentage for each to be used to create ads.

Maximum IDs per call: 500

requests. bidPercentage string Optional The bid percentage associated with listing. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. If bid percentage is not provided, the default bid percentage associated with the campaign is used. The higher the bid, the better the position of the listing on the page.

The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20




requests. listingId string Optional The eBay listing identifier. This is generated when the listing was created.

Maximum: 500
