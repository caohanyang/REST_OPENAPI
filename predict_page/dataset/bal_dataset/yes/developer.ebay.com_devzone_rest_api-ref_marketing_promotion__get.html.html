Get Promotions

GET /promotion


This call retrieves all the seller's undeleted promotions currently available on a specific eBay marketplace. The marketplace_id value is required, which limits the response to that specific site.

You can also sort the results and limit the response by the number of promotions to return, the promotion state, by eBay marketplace, and by matching a keyword query.

One way you might use this call is to populate a dashboard on the client side so the seller can perform further actions on a given promotion.

Maximum Returned: 200

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/marketing/v1/promotion?
  limit=string&
  marketplace_id=string&
  offset=string&
  promotion_status=string&
  q=string&
  sort=SortField

URI parameters

Parameter Type Required? Meaning
limit string Optional Specifies the number of promotions returned in a result set.

Default: 50
Maximum: 200


marketplace_id string Required The eBay marketplace Id of the site where the promotion is hosted. Valid values are EBAY_US, EBAY_DE, EBAY_FR, EBAY_IT, EBAY_ES, EBAY_AU, or EBAY_GB.
offset string Optional Specifies the number of promotions to skip in the result set. This is used with the limit field to control the pagination of the output.

If offset is 10 and limit is 10, the call will retrieve promotions 11 thru 20 from the list of promotions returned.

Default: 0



promotion_status string Optional Specifies to limit the results by the state of the promotion.

Valid Values:
SCHEDULED
RUNNING
PAUSED
DRAFT
ENDED
INVALID







q string Optional A string consisting of one or more words that are used to search for promotions.

For example: "iphone" or "Harry Potter."

When keywords are separated by a comma, the comma is ignored. So a keyword of "iphone, ipad" is the same as using "iphone ipad." Both return items that have iphone or ipad.



sort string Optional Specifies the order in which to sort the response.

Valid Values :
START_DATE
END_DATE
PROMOTION_NAME

If a value is proceeded by a dash, then it will sort in reverse order.

For example:
sort=endDate - means sort results by the promotion end date in ascending order
sort=-promotionName - means sort by the promotion name in descending order

Applicable values are from SortField .

Sample: Get All Promotions

Retrieves all the seller's promotions.

Description

This sample retrieves all the seller's promotions for a specific eBay marketplace.

Input

Only the marketplace_id parameter is required. There are several filters you can use to limit the response. This sample uses the limit , offset , promotion_status and sort filters. 

URL format. See also the non-wrapped version of this URL.

GET https://api.ebay.com/sell/marketing/v1/promotion?
   limit=5&
   offset=0&
   promotion_status=DRAFT&
   sort=START_DATE&
   marketplace_id=EBAY_US




Output

The output is all the seller's draft promotions, sorted by start date, and returns 5 promotions starting with the first one retrieved.
