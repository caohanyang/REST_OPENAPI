UpdateQuantity

Type that defines the fields needed to update the quantity of a line item.

Calls that use UpdateQuantity:

    /checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST

Fields

lineItemId ( string ) [0..?]
A unique eBay-assigned Id value to identify the line item in a purchase order.
See the Field Index to learn which calls use lineItemId.

quantity ( integer ) [0..?]
The number of individual items ordered in this line item, as specified by the buyer.
See the Field Index to learn which calls use quantity.
