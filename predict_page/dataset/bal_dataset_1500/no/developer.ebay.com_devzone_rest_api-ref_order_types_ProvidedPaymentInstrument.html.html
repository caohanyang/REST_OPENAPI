ProvidedPaymentInstrument

Type that defines the fields for reference information about the payment method. This is returned only if you have used the Update Payment Info call to change the payment method.

Type that uses ProvidedPaymentInstrument:

    CheckoutSessionResponse

Calls that use ProvidedPaymentInstrument:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

paymentInstrumentReference ( PaymentInstrumentReference ) [0..?]
Container for the payment reference, such as last four digits of a credit card.
See the Field Index to learn which calls use paymentInstrumentReference.

paymentMethodBrand ( PaymentMethodBrand ) [0..?]
Container for the name and logo of the credit card company (brand).
See the Field Index to learn which calls use paymentMethodBrand.

paymentMethodType ( PaymentMethodTypeEnum ) [0..?]
Indicates the method of payment, such as credit card.
See the Field Index to learn which calls use paymentMethodType.
