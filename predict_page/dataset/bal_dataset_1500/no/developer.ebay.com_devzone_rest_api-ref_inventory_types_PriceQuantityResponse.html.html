PriceQuantityResponse

This type is used to display the result for each offer that the seller attempted to update with a Bulk Update Price and Quantity call, as well as any errors or warnings that may have occurred while trying to update the price and/or quantity of that offer.

Type that uses PriceQuantityResponse:

    BulkPriceQuantityResponse

Call that uses PriceQuantityResponse:

    /bulk_update_price_quantity POST

Fields

errors ( array of ErrorDetailV3 )
offerId ( string )
sku ( string )
statusCode ( integer )
warnings ( array of ErrorDetailV3 )


errors ( array of ErrorDetailV3 ) [0..?]
This container will be returned if there were one or more errors associated with the update to the offer specified by the offerId value.
See the Field Index to learn which calls use errors.

offerId ( string ) [0..?]
The unique identifier of the offer that was updated.
See the Field Index to learn which calls use offerId.

sku ( string ) [0..?]
This is the seller-defined SKU value of the product in the offer that was updated.
See the Field Index to learn which calls use sku.

statusCode ( integer ) [0..?]
The value returned in this container will indicate the status of the attempt to update the price and/or quantity of the offer specified in the corresponding offerId field. For a completely successful update of an offer, a value of 200 will appear in this field. For a partially successful call, a value of 207 will appear in this field. A value of 204 will appear in this field if the response payload has no content.
See the Field Index to learn which calls use statusCode.

warnings ( array of ErrorDetailV3 ) [0..?]
This container will be returned if there were one or more warnings associated with the update to the offer specified by the offerId value.
See the Field Index to learn which calls use warnings.
