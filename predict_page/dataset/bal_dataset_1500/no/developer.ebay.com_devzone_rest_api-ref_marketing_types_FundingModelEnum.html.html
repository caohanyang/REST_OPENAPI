FundingModelEnum

Enumeration value representing the funding model (event that incurs a fee to the seller) of the campaign. Currently, only COST_PER_SALE is supported.

Type that uses FundingModelEnum:

    FundingStrategy

Calls that use one or more values of FundingModelEnum:

    /ad_campaign POST , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET , /ad_campaign/{campaign_id}/clone POST

Enumeration Value

Value Description
COST_PER_SALE Indicates the seller is charged for a promoted listing when the listing sells.
