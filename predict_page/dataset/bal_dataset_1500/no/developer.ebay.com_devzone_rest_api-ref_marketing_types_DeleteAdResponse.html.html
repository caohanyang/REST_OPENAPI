DeleteAdResponse

Type that defines the fields for the delete ad response.

Type that uses DeleteAdResponse:

    BulkDeleteAdResponse

Call that uses DeleteAdResponse:

    /ad_campaign/{campaign_id}/bulk_delete_ads_by_listing_id POST

Fields

adId ( string ) [0..?]
The identifier of the ad that was deleted.
See the Field Index to learn which calls use adId.

errors ( array of ErrorDetailV3 ) [0..?]
An array of the errors associated with this listing ID.
See the Field Index to learn which calls use errors.

listingId ( string ) [0..?]
The eBay listing ID of the deleted ad.
See the Field Index to learn which calls use listingId.

statusCode ( integer ) [0..?]
Indicates if the ad was deleted successfully.
See the Field Index to learn which calls use statusCode.
