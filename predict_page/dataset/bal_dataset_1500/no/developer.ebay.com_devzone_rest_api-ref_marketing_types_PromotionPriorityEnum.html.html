PromotionPriorityEnum

Enumerated type that defines the values for the priority (precedence) level. This is used to assign the precedence of a promotion.

Types that use PromotionPriorityEnum:

    ItemPromotion , ItemPromotionResponse , PromotionDetail

Calls that use one or more values of PromotionPriorityEnum:

    /item_promotion POST , /item_promotion/{promotion_id} GET , /promotion GET

Enumeration Values

Value Description
INVALID Indicates the value submitted was invalid. Correct the request and submit the call again.
PRIORITY_1 Indicates the rule ranks first in priority.
PRIORITY_2 Indicates the rule ranks second in priority.
PRIORITY_3 Indicates the rule ranks third in priority.
PRIORITY_4 Indicates the rule ranks fourth in priority.
* See the Enumeration Index to see exact use of each enumeration value in the API.
