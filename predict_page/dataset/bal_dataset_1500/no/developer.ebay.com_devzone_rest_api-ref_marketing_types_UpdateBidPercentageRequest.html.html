UpdateBidPercentageRequest

Type that defines the fields for the update bid percentage request.

Call that uses UpdateBidPercentageRequest:

    /ad_campaign/{campaign_id}/ad/{ad_id}/update_bid POST

Field

bidPercentage ( string ) [0..?]
The new bid percentage. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20


See the Field Index to learn which calls use bidPercentage.
