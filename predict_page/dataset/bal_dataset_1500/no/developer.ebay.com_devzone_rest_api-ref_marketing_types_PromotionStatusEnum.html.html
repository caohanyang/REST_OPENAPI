PromotionStatusEnum

Enumerated type that defines the values for the state of a promotion.

Types that use PromotionStatusEnum:

    ItemPromotion , ItemPromotionResponse , PromotionDetail

Calls that use one or more values of PromotionStatusEnum:

    /item_promotion POST , /item_promotion/{promotion_id} GET , /promotion GET

Enumeration Values

Value Description
DRAFT Indicates the promotion is a draft. To schedule or make the promotion active use the Create Item Promotion call to submit the full details of the promotion.
ENDED Indicates the promotion has ended.
INVALID This is for internal use only for when promotion type specified in the request is invalid.
PAUSED Indicates the promotion is currently on hold. To restart the promotion use the Resume Promotion call
RUNNING Indicates the promotion is live on the site.
SCHEDULED Indicates the promotion start date is in future (greater than current date). The promotion is not live yet.
* See the Enumeration Index to see exact use of each enumeration value in the API.
