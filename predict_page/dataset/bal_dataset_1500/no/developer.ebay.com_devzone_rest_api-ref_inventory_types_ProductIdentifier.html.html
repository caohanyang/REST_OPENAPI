ProductIdentifier

This type is used to identify a motor vehicle that is compatible with the corresponding inventory item. The motor vehicle can be identified through an eBay Product ID, a K type value, or a Global Trade Identification Number. If the specified motor vehicle is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.
This type is only applicable to the Create Or Replace Product Compatibility call, and will not be returned in the Get Product Compatibility call.

Type that uses ProductIdentifier:

    CompatibleProduct

Call that uses ProductIdentifier:

    /inventory_item/{sku}/product_compatibility PUT

Fields

epid ( string ) [0..?]
This field can be used if the seller already knows the eBay product ID associated with the motor vehicle that is to be added to the compatible product list. If this eBay product ID is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.
See the Field Index to learn which calls use epid.

gtin ( string ) [0..?]
This field can be used if the seller knows the Global Trade Identification Number for the motor vehicle that is to be added to the compatible product list. If this GTIN value is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.
See the Field Index to learn which calls use gtin.

ktype ( string ) [0..?]
This field can be used if the seller knows the K Type Number for the motor vehicle that is to be added to the compatible product list. If this K Type value is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.

Only the DE, UK, and AU sites support the use of K Type Numbers.
See the Field Index to learn which calls use ktype.
