SummaryReportResponse

Type that defines the fields for the promotion summary report. This report is returned in JSON. For details about this report, see Promotion Summary Report Details .

Call that uses SummaryReportResponse:

    /promotion_summary_report GET

Fields

baseSale ( Amount )
lastUpdated ( string )
percentageSalesLift ( string )
promotionSale ( Amount )
totalSale ( Amount )


baseSale ( Amount ) [0..?]
The total revenue from all the purchased items that were part of a promotion, but did not trigger a discount during the promotion period.
See the Field Index to learn which calls use baseSale.

lastUpdated ( string ) [0..?]
The date the report was generated.
See the Field Index to learn which calls use lastUpdated.

percentageSalesLift ( string ) [0..?]
Represents the percentage of the total dollar amount gained due to promotions. precentageSalesLift = promotionSale / ( baseSale + promotionSale ).
See the Field Index to learn which calls use percentageSalesLift.

promotionSale ( Amount ) [0..?]
The total revenue from all the purchased items that were part of a promotion and did trigger a discount during the promotion period.
See the Field Index to learn which calls use promotionSale.

totalSale ( Amount ) [0..?]
Total dollar amount in sales of all the seller's listing as of the date the report was generated.
See the Field Index to learn which calls use totalSale.
