InventoryReferenceTypeEnum

Type that uses InventoryReferenceTypeEnum:

    InventoryItem

Calls that use one or more values of InventoryReferenceTypeEnum:

    /item_promotion POST , /item_promotion/{promotion_id} GET

Enumeration Values

Value Description
INVENTORY_ITEM Indicates the inventory reference ID is for a listing.
INVENTORY_ITEM_GROUP Indicates the inventory reference ID is the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors.
