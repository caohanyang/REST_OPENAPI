LineItem

Container for an individual line item in a checkout transaction.

Type that uses LineItem:

    CheckoutSessionResponse

Calls that use LineItem:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

baseUnitPrice ( Amount ) [0..?]
The cost (Buy It Now price) of a single item in this line item. It is a starting point for computing the price during checkout session.
See the Field Index to learn which calls use baseUnitPrice.

image ( Image ) [0..?]
An eBay-assigned URL that points to an image of the item. eBay assigns the URL when seller uploads the image.
See the Field Index to learn which calls use image.

itemId ( string ) [0..?]
A unique eBay-assigned Id value that identifies the item.
See the Field Index to learn which calls use itemId.

lineItemId ( string ) [0..?]
A unique eBay-assigned Id value that identifies the line item in a checkout session.
See the Field Index to learn which calls use lineItemId.

netPrice ( Amount ) [0..?]
The total cost for the items in this line item taking into account the quantity and applying any seller item discounts, such as Buy 1 Get 1. Note : This also does not include any shipping discounts, shipping costs, fees, or seller adjustments.
See the Field Index to learn which calls use netPrice.

promotions ( array of Promotion ) [0..?]
An array of promotions associated with the item of this line item.
See the Field Index to learn which calls use promotions.

quantity ( integer ) [0..?]
The number of individual items ordered for this line item, as specified by the buyer.
See the Field Index to learn which calls use quantity.

seller ( Seller ) [0..?]
Container for the seller's eBay user name and their logo.
See the Field Index to learn which calls use seller.

shippingOptions ( array of ShippingOption ) [0..?]
An array of the shipping methods that are available for the line item. By default, the first one will be selected.
See the Field Index to learn which calls use shippingOptions.

shortDescription ( string ) [0..?]
An abbreviated version of the item description.
See the Field Index to learn which calls use shortDescription.

title ( string ) [0..?]
The seller created title of the item.
See the Field Index to learn which calls use title.
