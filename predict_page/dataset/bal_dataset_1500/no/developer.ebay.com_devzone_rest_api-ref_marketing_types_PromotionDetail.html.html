PromotionDetail

Type the defines the field for the details of a promotion.

Type that uses PromotionDetail:

    PromotionsPagedCollection

Call that uses PromotionDetail:

    /promotion GET

Fields

description ( string )
endDate ( string )
marketplaceId ( MarketplaceIdEnum )
name ( string )
priority ( PromotionPriorityEnum )
promotionHref ( string )
promotionId ( string )
promotionImageUrl ( string )
promotionStatus ( PromotionStatusEnum )
startDate ( string )


description ( string ) [0..?]
The description of a promotion.
See the Field Index to learn which calls use description.

endDate ( string ) [0..?]
The date the promotion ends.
See the Field Index to learn which calls use endDate.

marketplaceId ( MarketplaceIdEnum ) [0..?]
The eBay marketplace Id of the site where the promotion is hosted.
See the Field Index to learn which calls use marketplaceId.

name ( string ) [0..?]
The name of the promotion.
See the Field Index to learn which calls use name.

priority ( PromotionPriorityEnum ) [0..?]
Indicates the precedence of the promotion. This is used to determine the position of a promotion on the seller's All Offers page.
See the Field Index to learn which calls use priority.

promotionHref ( string ) [0..?]
The resource location for the promotion.
See the Field Index to learn which calls use promotionHref.

promotionId ( string ) [0..?]
The identifier of the promotion. This ID is generated when the promotion was created.
See the Field Index to learn which calls use promotionId.

promotionImageUrl ( string ) [0..?]
The URL of the image used to represent the promotion on the seller's All Offer page.
See the Field Index to learn which calls use promotionImageUrl.

promotionStatus ( PromotionStatusEnum ) [0..?]
Indicates the state of the promotion.

Note: When creating or updating a promotion the only valid status values are DRAFT and SCHEDULED.

See the Field Index to learn which calls use promotionStatus.

startDate ( string ) [0..?]
The start date of the promotion in UTC format (i.e., YYYY-MM-DDTHH:MM:SS.SSSZ).
See the Field Index to learn which calls use startDate.
