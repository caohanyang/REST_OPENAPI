CreditCard

Type that defines the fields for the credit card that's used to pay for the items.

Types that use CreditCard:

    CreateGuestCheckoutSessionRequest , CreateSignInCheckoutSessionRequest , UpdatePaymentInformation

Calls that use CreditCard:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST

Fields

accountHolderName ( string )
billingAddress ( BillingAddress )
brand ( string )
cardNumber ( string )
cvvNumber ( string )
expireMonth ( integer )
expireYear ( integer )


accountHolderName ( string ) [0..?]
The name of the card holder under which the credit card was issued.

Required when submitting credit card information.

See the Field Index to learn which calls use accountHolderName.

billingAddress ( BillingAddress ) [0..?]
Container for the billing address of the card holder. Note: If the address cannot be validated, a warning message will be returned.

Required when submitting credit card information.

See the Field Index to learn which calls use billingAddress.

brand ( string ) [0..?]
The type of the credit card (Visa, MasterCard, AmEx, Discover, etc.).

Required when submitting credit card information.

See the Field Index to learn which calls use brand.

cardNumber ( string ) [0..?]
The credit card number on the card.

Required when submitting credit card information.

See the Field Index to learn which calls use cardNumber.

cvvNumber ( string ) [0..?]
The Card Verification Value of the credit card. This value is also known as the card verification code (CVC) or card security code (CSC). This is a three-digit number on VISA®, MasterCard®, and Discover® branded credit and debit cards. On American Express® branded cards, this is a four-digit numeric code.

Note: This number is not the PIN associated with the card.

Required when submitting credit card information.



See the Field Index to learn which calls use cvvNumber.

expireMonth ( integer ) [0..?]
The month the credit card expires.

Required when submitting credit card information.

See the Field Index to learn which calls use expireMonth.

expireYear ( integer ) [0..?]
The year the credit card expires.

Required when submitting credit card information.

See the Field Index to learn which calls use expireYear.
