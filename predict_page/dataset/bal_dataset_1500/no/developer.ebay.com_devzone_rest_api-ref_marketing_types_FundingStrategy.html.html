FundingStrategy

Type that defines the fields for the funding strategy.

Types that use FundingStrategy:

    Campaign , CloneCampaignRequest , CreateCampaignRequest

Calls that use FundingStrategy:

    /ad_campaign POST , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET , /ad_campaign/{campaign_id}/clone POST

Fields

bidPercentage ( string ) [0..?]
The default bid percentage for the funding model. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. If bid percentage is not provided, the default bid percentage associated with the campaign is used. The higher the bid, the better the position of the listing on the page.

The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20




See the Field Index to learn which calls use bidPercentage.

fundingModel ( FundingModelEnum ) [0..?]
Indicates the funding model associated with the campaign.

Default: COST_PER_SALE.

See the Field Index to learn which calls use fundingModel.
