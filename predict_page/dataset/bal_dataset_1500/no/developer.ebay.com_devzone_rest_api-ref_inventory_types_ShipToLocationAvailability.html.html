ShipToLocationAvailability

This type is used to specify the quantity of the inventory item that is available for standard shipping to the buyer's location.

Types that use ShipToLocationAvailability:

    Availability , PriceQuantity

Calls that use ShipToLocationAvailability:

    /bulk_update_price_quantity POST , /inventory_item GET , /inventory_item/{sku} PUT

Field

quantity ( integer ) [0..?]
Required if the original Create or Replace Inventory Item call to create the inventory item had 'ship-to-location' inventory set up, and another Create or Replace Inventory Item call is being made to update the inventory item. This integer value indicates the quantity of the inventory item that is available for standard shipping to the buyer's location.
See the Field Index to learn which calls use quantity.
