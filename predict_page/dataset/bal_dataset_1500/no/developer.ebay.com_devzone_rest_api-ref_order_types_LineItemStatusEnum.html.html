LineItemStatusEnum

An enumeration type for the values that indicate the status of the items in a line item.

Type that uses LineItemStatusEnum:

    LineItem_0

Calls that use one or more values of LineItemStatusEnum:

    /guest_purchase_order/{purchaseOrderId} GET , /purchase_order/{purchaseOrderId} GET

Enumeration Values

Value Description
CANCELLED Indicates this line item has been cancelled.
DELIVERED Indicates this line item has been delivered.
FULFILLMENT_IN_PROGRESS Indicates this line item is in the process of being shipped or in transit.
PENDING Indicates this line item has not been shipped.
* See the Enumeration Index to see exact use of each enumeration value in the API.
