PurchaseOrderSummary

Type that defines the fields for the summary of a purchase order.

Calls that use PurchaseOrderSummary:

    /checkout_session/{checkoutSessionId}/place_order POST , /guest_checkout_session/{checkoutSessionId}/place_order POST

Fields

purchaseOrderHref ( string ) [0..?]
A HATEOAS reference of the purchase order.
See the Field Index to learn which calls use purchaseOrderHref.

purchaseOrderId ( string ) [0..?]
A unique identifier of the purchase order. When a checkout session completes, a purchase order Id is generated.
See the Field Index to learn which calls use purchaseOrderId.

purchaseOrderPaymentStatus ( PurchaseOrderPaymentStatusEnum ) [0..?]
Indicates the payment status for the purchase order.
See the Field Index to learn which calls use purchaseOrderPaymentStatus.

warnings ( array of ErrorDetailV3 ) [0..?]
An array of warning messages.
See the Field Index to learn which calls use warnings.
