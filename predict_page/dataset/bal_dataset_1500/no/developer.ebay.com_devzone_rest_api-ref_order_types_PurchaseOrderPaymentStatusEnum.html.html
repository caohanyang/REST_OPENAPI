PurchaseOrderPaymentStatusEnum

An enumerated type for the values that indicate the status of the purchase order.

Types that use PurchaseOrderPaymentStatusEnum:

    GuestPurchaseOrder , PurchaseOrder , PurchaseOrderSummary

Calls that use one or more values of PurchaseOrderPaymentStatusEnum:

    /checkout_session/{checkoutSessionId}/place_order POST , /guest_checkout_session/{checkoutSessionId}/place_order POST , /guest_purchase_order/{purchaseOrderId} GET , /purchase_order/{purchaseOrderId} GET

Enumeration Values

Value Description
FAILED Indicates that processing the payment for the purchase order has failed.
PAID Indicates the purchase order has been paid for.
PARTIALLY_PAID Indicates that only some of the line items in the purchase order have been paid for.
PENDING Indicates the payment for the purchase order is pending.
