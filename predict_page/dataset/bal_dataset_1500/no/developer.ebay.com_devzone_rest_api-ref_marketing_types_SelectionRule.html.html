SelectionRule

Type that defines the fields for the selection rules used to create a campaign.

Type that uses SelectionRule:

    CampaignCriterion

Calls that use SelectionRule:

    /ad_campaign POST , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET

Fields

brands ( array of string )
categoryIds ( array of string )
categoryScope ( CategoryScopeEnum )
listingConditionIds ( array of string )
maxPrice ( Amount )
minPrice ( Amount )


brands ( array of string ) [0..?]
An array of the brands of the listings that are to be included in the campaign.
See the Field Index to learn which calls use brands.

categoryIds ( array of string ) [0..?]
An array of the category IDs of the listings that are to be included in the campaign.
See the Field Index to learn which calls use categoryIds.

categoryScope ( CategoryScopeEnum ) [0..?]
Indicates the source of the category ID; eBay or seller's store.
See the Field Index to learn which calls use categoryScope.

listingConditionIds ( array of string ) [0..?]
A comma separated list of condition IDs of the listings that are to be included in the campaign. For details see Item Condition IDs .

For example: 1000,1500

See the Field Index to learn which calls use listingConditionIds.

maxPrice ( Amount ) [0..?]
The maximum price of the listings that are to be included in the campaign.
See the Field Index to learn which calls use maxPrice.

minPrice ( Amount ) [0..?]
The minimum price of the listings that are to be included in the campaign.
See the Field Index to learn which calls use minPrice.
