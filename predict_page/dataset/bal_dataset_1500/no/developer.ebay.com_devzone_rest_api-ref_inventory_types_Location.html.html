Location

A complex type that defines a a physical address location with GeoCoordinates.

Types that use Location:

    InventoryLocationFull , InventoryLocationResponse

Calls that use Location:

    /location GET , /location/{merchantLocationKey} POST

Fields

address ( Address ) [0..?]
The address container is required for a Create Inventory Location call, and it is always returned in Get Inventory Location / Get Inventory Locations calls. Except in the case of an inventory location that supports In-Store Pickup or Click and Collect inventory, a full address is not a requirement when setting up an inventory location.
See the Field Index to learn which calls use address.

geoCoordinates ( GeoCoordinates ) [0..?]
This container is used to display the Global Positioning System (GPS) latitude and longitude coordinates for the inventory location.
See the Field Index to learn which calls use geoCoordinates.
