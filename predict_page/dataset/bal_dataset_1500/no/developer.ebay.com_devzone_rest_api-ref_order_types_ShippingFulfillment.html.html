ShippingFulfillment

Type that defines the fields for the shipping details.

Type that uses ShippingFulfillment:

    PurchaseOrder

Call that uses ShippingFulfillment:

    /purchase_order/{purchaseOrderId} GET

Fields

actualDeliveryDate ( string )
lineItemReferences ( array of LineItemReference )
maxEstimatedDeliveryDate ( string )
minEstimatedDeliveryDate ( string )
shipmentTrackingNumber ( string )
shipmentTrackingUrl ( string )
shippedDate ( string )
shippingCarrierName ( string )
shippingServiceName ( string )


actualDeliveryDate ( string ) [0..?]
The date the purchase order was delivered.
See the Field Index to learn which calls use actualDeliveryDate.

lineItemReferences ( array of LineItemReference ) [0..?]
Type the defines the fields for the line item Id and the quantity.
See the Field Index to learn which calls use lineItemReferences.

maxEstimatedDeliveryDate ( string ) [0..?]
The end of the date range in which the purchase order is expected to be delivered to the shipping address.
See the Field Index to learn which calls use maxEstimatedDeliveryDate.

minEstimatedDeliveryDate ( string ) [0..?]
The beginning of the date range in which the purchase order is expected to be delivered to the shipping address.
See the Field Index to learn which calls use minEstimatedDeliveryDate.

shipmentTrackingNumber ( string ) [0..?]
The shipping provider number associated with the purchase order that can be used to track the package.
See the Field Index to learn which calls use shipmentTrackingNumber.

shipmentTrackingUrl ( string ) [0..?]
The URL of the shipping provider's shipment tracking page.
See the Field Index to learn which calls use shipmentTrackingUrl.

shippedDate ( string ) [0..?]
The date the seller gave the purchase order to the shipping provider.
See the Field Index to learn which calls use shippedDate.

shippingCarrierName ( string ) [0..?]
A name of the shipping provider, such as FedEx, or USPS.
See the Field Index to learn which calls use shippingCarrierName.

shippingServiceName ( string ) [0..?]
A name of a shipping type. For example, Priority Mail Express (provided by USPS) or FedEx International Priority (Provided by FedEx).
See the Field Index to learn which calls use shippingServiceName.
