Get Purchase Order

GET /purchase_order/{purchaseOrderId}

Retrieves the details about a specific eBay member purchase order and its line items, including purchase order status; dates created and modified; item quantity and listing data; payment and shipping information; and prices, taxes, and discounts and credits.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/buy/order/v1/purchase_order/{purchaseOrderId}

URI parameters

Parameter Type Required? Meaning
purchaseOrderId string Required The unique identifier of a purchase order made by an eBay Member, for which details are to be retrieved. This value is returned in the purchaseOrderId field by the Checkout API's Place Order and Place Guest Order calls.


HTTP request headers

All requests made to eBay REST operations require you to provide the authorization HTTP header for authentication.
See HTTP request headers for details.



OAuth request scope

This request requires a user access token with the following scope:

https://api.ebay.com/oauth/api_scope/buy.order.readonly

https://api.ebay.com/oauth/api_scope/buy.order

See Getting Access Tokens for more information.

This call has no request payload.
