Get Sales Tax Jurisdictions

GET /country/{countryCode}/sales_tax_jurisdiction

This call retrieves all the sales tax jurisdictions for the country that you specify in the { countryCode } path parameter. Countries with valid sales tax jurisdictions are Canada, India, and the US.

The response from this call tells you the jurisdictions for which a seller can configure tax tables. Although setting up tax tables is optional, you can use the Create Sales Tax Tables in the Account API call to configure the tax tables for the jurisdictions you sell to.

Resource URI (production) 

GET https://api.ebay.com/sell/metadata/v1/country/{countryCode}/sales_tax_jurisdiction

URI parameters

Parameter Type Required? Meaning
countryCode string Required A two-letter ISO 3166-1 Alpha-2 country code. eBay provides sales tax jurisdiction information for Canada, India, and the United States.Valid values for this path parameter are CA , IN , and US .

Applicable values are from CountryCodeEnum .

Sample: Get the sales tax jurisdictions for Canada.

This call retrieves the sales tax jurisdictions for the country you specify.

Description

This call retrieves the sales tax jurisdictions for a country. However, not all countries have multiple sales tax jurisdictions. Current support is for Canada , India , and the United States .

Input

This call does not use a payload. Supply the country code as a path parameter. The example below specified Canada (CA). 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/metadata/v1/country/CA/sales_tax_jurisdiction

Output

If the call is successful, it returns all the sales tax jurisdictions for the specified country.
