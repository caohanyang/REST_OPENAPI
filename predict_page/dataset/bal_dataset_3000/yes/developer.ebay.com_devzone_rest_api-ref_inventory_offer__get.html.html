Get Offers

GET /offer

This call retrieves all existing offers for the specified SKU value. The seller has the option of limiting the offers that are retrieved to a specific eBay marketplace, or to a listing format.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/inventory/v1/offer?
  sku=string&
  marketplace_id=MarketplaceEnum&
  format=FormatTypeEnum&
  limit=string&
  offset=string

URI parameters

Parameter Type Required? Meaning
format string Optional This enumeration value sets the listing format for the offer. This query parameter will be passed in if the seller only wants to see product offers in this listing format. For the initial version of the Inventory API, only fixed price is supported.

Applicable values are from FormatTypeEnum .
limit string Optional The value passed in this query parameter sets the maximum number of records to return per page of data. This value must be an integer value from 1 to 100 . If this query parameter is not set, its value defaults to 100 records per page.
marketplace_id string Optional The unique identifier of the eBay marketplace. This query parameter will be passed in if the seller only wants to see the product's offers on a specific eBay marketplace.

Applicable values are from MarketplaceEnum .
offset string Optional The value passed in this query parameter sets the page number to retrieve. The first page of records has a value of 0 , the second page of records has a value of 1 , and so on. If this query parameter is not set, its value defaults to 0 , and the first page of records is returned.
sku string Required The seller-defined SKU value is passed in as a query parameter. All offers associated with this product are returned in the response.

Max length : 50.

Sample: Retrieves Multiple Offers for an Inventory Item

This call retrieves multiples offer for an inventory item.

Description

This call will retrieve multiple offers for a specific inventory item associated with the seller's acount.

Input

The seller is looking for any and all offers for the inventory item specified in the sku query parameter. All offers associated with this inventroy item will be retrieved. 

URL format. See also the non-wrapped version of this URL.

GET https://api.ebay.com/sell/inventory/v1/offer?

Two offers are retrieved for inventory item with a SKU value of 3455632452345 .

The first retrieved offer is in the published state, and there is an active eBay listing on the eBay US site associated with this offer. The item ID, status, and quantity sold for this listing is shown in the listing container. There is still a quantity of 70 available for purchase.
