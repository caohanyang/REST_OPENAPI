Get Promotion Summary Report

GET /promotion_summary_report

This call generates a summary report for all the seller's undeleted promotions (RUNNING, PAUSED, ENDED) for the specified eBay marketplace. It summarizes the seller's campaign performance for all promotions on a given site.

For details about this report, see Promotion Summary Report Details.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/marketing/v1/promotion_summary_report?
  marketplace_id=string

URI parameters

Parameter Type Required? Meaning
marketplace_id string Required The eBay marketplace Id of the site you for which you want a promotion summary report. Valid values are EBAY_US, EBAY_DE, EBAY_FR, EBAY_IT, EBAY_ES, EBAY_AU, or EBAY_GB.


HTTP request headers

All requests made to eBay REST operations require you to provide the authorization HTTP header for authentication.
