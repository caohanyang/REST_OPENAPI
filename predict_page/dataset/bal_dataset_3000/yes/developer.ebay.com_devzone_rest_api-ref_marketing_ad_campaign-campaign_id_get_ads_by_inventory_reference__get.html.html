Get Ads By Inventory Reference

GET /ad_campaign/{campaign_id}/get_ads_by_inventory_reference


This call retrieves ads from the specified campaign using the seller's inventory reference ID and type. A seller's inventory reference ID is the identifier of a listing or the identifier of the parent of a listing with variations, such as a shirt that is available in multiple sizes and colors.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/marketing/v1/ad_campaign/{campaign_id}/get_ads_by_inventory_reference?
  inventory_reference_id=string&
  inventory_reference_type=string

URI parameters

Parameter Type Required? Meaning
campaign_id string Required Identifier of a campaign in which the ads you want returned are associated. This ID was generated when the campaign was created.
inventory_reference_id string Required The seller's inventory reference ID associated with the ad you want returned. A seller's inventory reference ID is the identifier of either a listing or the identifier of the parent of a listing with variations, such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .
inventory_reference_type string Required The type of the seller's inventory ID, which is a listing or group of items. You must always pass in both inventory_reference_id and inventory_reference_type .

Response field descriptions



Output Container/Field Type Occurrence Meaning
ads array of Ad Conditionally An array of ads based on their inventory reference.
ads. adId string Conditionally Identifier of an ad. This ID was generated when the ad was created.
ads. bidPercentage string Conditionally The bid percentage associated with the listing. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0
Maximum value: 20


ads. inventoryReferenceId string Conditionally The seller's inventory reference ID for a listing. An inventory reference is either the ID of a listing or the ID of the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors. You must always pass in both inventory_reference_id and inventory_reference_type .
ads. inventoryReferenceType string Conditionally Indicates the inventory reference ID type of the inventoryReferenceId . You must always pass in both inventory_reference_id and inventory_reference_type .

Applicable values are from InventoryReferenceTypeEnum :

INVENTORY_ITEM
Indicates the inventory reference ID is for a listing.
INVENTORY_ITEM_GROUP
Indicates the inventory reference ID is the parent of a listing with variations (a group of items), such as a shirt that is available in multiple sizes and colors.

Code so that your app gracefully handles any future changes to this list.
ads. listingId string Conditionally The eBay listing identifier. This is generated when the listing was created.

Sample: Retrieve an Ad by Inventory Reference ID

Retrieves an ad in a specific campaign using the seller's inventory reference ID of the listing.
