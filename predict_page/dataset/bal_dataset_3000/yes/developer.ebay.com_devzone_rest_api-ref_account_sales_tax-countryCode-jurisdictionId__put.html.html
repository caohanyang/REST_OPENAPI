Create a Sales Tax Table Entry

PUT /sales_tax/{countryCode}/{jurisdictionId}

Use this call to create or update a sales tax table entry for a jurisdiction. Specify the tax table entry you want to configure using the two path parameters: countryCode and jurisdictionId . A tax table entry for a jurisdiction is comprised of two fields: one for the jurisdiction's sales-tax rate and another that's a boolean value indicating whether or not shipping and handling are taxed in the jurisdiction.

You can set up tax tables for countries that support differnt tax jurisdictions . Currently, only Canada, India, and the US support separate tax jurisdictions. If you sell into any of these countries, you can set up tax tables for any of the country's jurisdictions. Retrieve valid jurisdiction IDs using Get Sales Tax Jurisdictions in the Metadata API.

Resource URI (production) 

PUT https://api.ebay.com/sell/account/v1/sales_tax/{countryCode}/{jurisdictionId}

URI parameters

Parameter Type Required? Meaning
countryCode string Required A two-letter ISO 3166-1 Alpha-2 .

Applicable values are from CountryCodeEnum .
jurisdictionId string Required Unique ID for a sales tax jurisdiction.

{ /* SalesTaxBase */

}

Request field descriptions



Input Container/Field Type Occurrence Meaning
salesTaxPercentage string Required The sales tax rate, as a percentage of the sale.
shippingAndHandlingTaxed boolean Optional Set this value to true to indicate that shipping and handling charges are taxed.

Sample: Get a sales tax table.

This call returns the sales tax table for the specified sales tax jurisdiction.

Description

This call returns a sales tax table for the specified country and sales tax jurisdiction. The seller must have previously configured the sale-tax table, otherwise the response will contain an empty list.

Input

This example gets the sales tax table for New York, a sales tax jurisdiction in the United States. 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/account/v1/sales_tax/US/NY

{
    "salesTaxPercentage": 4.4,
    "shippingAndHandlingTaxed": true
}

Output

A successful call returns no payload. 

JSON format.
If successful, this call returns an HTTP status code of "204 No content" and an empty payload.
