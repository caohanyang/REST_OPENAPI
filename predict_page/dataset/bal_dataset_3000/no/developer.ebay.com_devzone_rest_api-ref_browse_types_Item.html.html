Item

Type that defines the item details for a specific item.

Call that uses Item:

    /item/{item_id} GET

additionalImages ( array of Image ) [0..?]
This container is an array of URLs for Gallery Plus images of the item listing.
See the Field Index to learn which calls use additionalImages.

ageGroup ( string ) [0..?]
(Item Aspect) The age group for which the product is recommended. For example, newborn, infant, toddler, kids, adult, etc. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use ageGroup.

availabilityStatusForShipToHome ( AvailabilityStatusEnum ) [0..?]
The availability of this item.
See the Field Index to learn which calls use availabilityStatusForShipToHome.

bidCount ( integer ) [0..?]
This integer value indicates the total number of bids that have been placed against an auction item. This field will only be returned for auction listings.
See the Field Index to learn which calls use bidCount.

brand ( string ) [0..?]
(Item Aspect) The name brand of the item, such as Nike, Apple, etc. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use brand.

buyingOptions ( array of string ) [0..?]
Buying options information
See the Field Index to learn which calls use buyingOptions.

categoryPath ( string ) [0..?]
Text that shows the category hierarchy of the item. For example: Computers/Tablets & Networking, Laptops & Netbooks, PC Laptops & Netbooks
See the Field Index to learn which calls use categoryPath.

color ( string ) [0..?]
(Item Aspect) Text describing the color of the item. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use color.

condition ( string ) [0..?]
Text describing the condition of the item.
See the Field Index to learn which calls use condition.

currentBidPrice ( Amount ) [0..?]
This container displays the current highest bid for an auction item. The value (string) field shows the dollar value of the current highest bid, and the currency (3-digit ISO code) field denotes the currency associated with that bid value. This container will only be returned for auction listings.
See the Field Index to learn which calls use currentBidPrice.

description ( string ) [0..?]
The full description of the item that was created by the seller. This can be plain text or rich content.
See the Field Index to learn which calls use description.

energyEfficiencyClass ( string ) [0..?]
The enumeration value returned in this field indicates the energy efficiency rating of the item. Energy efficiency ratings apply to products listed by commercial vendors in electronics categories only. Currently, this field is only applicable for the Germany site, and this field will only be returned if the seller specified the energy efficiency rating through item specifics at listing time.
See the Field Index to learn which calls use energyEfficiencyClass.

gender ( string ) [0..?]
(Item Aspect) The gender for the item. This is used for items that could vary by gender, such as clothing. For example: male, female, or unisex. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use gender.

gtin ( string ) [0..?]
The unique Global Trade Item, (UPC, ISBN, EAN) number of the item as defined by http://www.gtin.info .
See the Field Index to learn which calls use gtin.

image ( Image ) [0..?]
The URL to the primary image of the item. The other images of the item are returned in the additionalImages field.
See the Field Index to learn which calls use image.

itemAffiliateWebUrl ( string ) [0..?]
This URL is the link to the View Item page with an affiliate tracking ID included. This field is only returned if the seller enables affiliate tracking for the listing. To enable affiliate tracking, the X-EBAY-C-ENDUSERCTX request header is required.
See the Field Index to learn which calls use itemAffiliateWebUrl.

itemEndDate ( string ) [0..?]
The date and time up to which the item(s) in the listing can be purchased.
See the Field Index to learn which calls use itemEndDate.

itemId ( string ) [0..?]
The unique identifier of the item.
Example: v1|150006693022|450002439078
See the Field Index to learn which calls use itemId.

itemLocation ( Address ) [0..?]
The physical location of the item.
See the Field Index to learn which calls use itemLocation.

localizedAspects ( array of TypedNameValue ) [0..?]
(Additional Item Aspects) A list of name/value pairs that describe the variations of the item, which have been changed to the language used by the site on which the item appears. For an example, see the variesByLocalizedAspects container.
See the Field Index to learn which calls use localizedAspects.

marketingPrice ( MarketingPrice ) [0..?]
The original price and the discount amount and percentage.
See the Field Index to learn which calls use marketingPrice.

material ( string ) [0..?]
(Item Aspect) Text describing what the item is made of. For example, silk. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use material.

mpn ( string ) [0..?]
The manufacturer's part number, which is a unique number that identifies a specific product.
See the Field Index to learn which calls use mpn.

pattern ( string ) [0..?]
(Item Aspect) Text describing the pattern used on the item. For example, paisley. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use pattern.

price ( Amount ) [0..?]
The cost of the item.
See the Field Index to learn which calls use price.

priceDisplayCondition ( PriceDisplayConditionEnum ) [0..?]
This enumeration value indicates how item price is displayed. For some items, the prices are so low that they can only be shown to the buyer after they add the item to the cart.
See the Field Index to learn which calls use priceDisplayCondition.

primaryItemGroupHref ( string ) [0..?]
A HATEOAS reference to the Item Group.
See the Field Index to learn which calls use primaryItemGroupHref.

primaryItemGroupId ( string ) [0..?]
The identifier of a group of items. These are all the same item that have various aspect differences, such as color, size, storage capacity, etc.
See the Field Index to learn which calls use primaryItemGroupId.

primaryProductReviewRating ( ReviewRating ) [0..?]
The product rating details.
See the Field Index to learn which calls use primaryProductReviewRating.

quantityLimitPerBuyer ( integer ) [0..?]
The maximum number for a specific item that one buyer can purchase.
See the Field Index to learn which calls use quantityLimitPerBuyer.

quantitySold ( integer ) [0..?]
The number of items that have been sold. This field is only applicable to multi-quantity, fixed-price listings.
See the Field Index to learn which calls use quantitySold.

returnTerms ( ItemReturnTerms ) [0..?]
This container provides an overview of the seller's return policy.
See the Field Index to learn which calls use returnTerms.

seller ( Seller ) [0..?]
The details about the seller of the item.
See the Field Index to learn which calls use seller.

shippingOptions ( array of ShippingOption ) [0..?]
An array of shipping options containers that have the details about cost, carrier, etc. of one shipping option. By default, the first shipping option is selected.
See the Field Index to learn which calls use shippingOptions.

shortDescription ( string ) [0..?]
A snippet of the full description of the item in plain text.
See the Field Index to learn which calls use shortDescription.

size ( string ) [0..?]
(Item Aspect) The size of the item. For example, '7' for a size 7 shoe. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use size.

sizeSystem ( string ) [0..?]
(Item Aspect) The sizing system of the country. Other aspects are returned in the localizedAspects container.

Valid Values:
US
UK
EU
DE
FR
JP
CN (China)
IT
BR
MEX

See the Field Index to learn which calls use sizeSystem.

sizeType ( string ) [0..?]
(Item Aspect) Text describing a size group in which the item would be included, such as regular, petite, plus, big-and-tall or maternity. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use sizeType.

subtitle ( string ) [0..?]
An additional shorten title for the item that is created by the seller.
See the Field Index to learn which calls use subtitle.

title ( string ) [0..?]
The seller-created title of the item.
See the Field Index to learn which calls use title.

topRatedBuyingExperience ( boolean ) [0..?]
This boolean field is returned as true for top-rated listings. As a buyer, three things that you get from a top-rated listing is a 30-day or longer return policy, same-day or 1-day handling time, and the added comfort of knowing that you are purchasing an item from a top-rated seller. See the Becoming a Top Rated Seller and qualifying for Top Rated Plus help topic for more information about Top Rated Seller and Top Rated Plus listing requirements.
See the Field Index to learn which calls use topRatedBuyingExperience.

uniqueBidderCount ( integer ) [0..?]
This integer value indicates the number of different eBay users who have placed one or more bids on an auction item. This field is only applicable to auction listings.
See the Field Index to learn which calls use uniqueBidderCount.

warnings ( array of ErrorDetailV3 ) [0..?]
An array of warning messages. These type of errors do not prevent the call from executing but should be checked.
See the Field Index to learn which calls use warnings.
