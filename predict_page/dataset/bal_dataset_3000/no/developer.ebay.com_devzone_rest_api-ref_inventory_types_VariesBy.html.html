VariesBy

This type is used to specify product aspects for which variations within an inventory item group vary.

Type that uses VariesBy:

    InventoryItemGroup

Call that uses VariesBy:

    /inventory_item_group/{inventoryItemGroupKey} PUT

Fields

aspectsImageVariesBy ( array of string ) [0..?]
This container will contain one product aspect where the variations of the inventory item group may vary, such as color or size.
See the Field Index to learn which calls use aspectsImageVariesBy.

specifications ( array of Specification ) [0..?]
This container consists of an array of one or more product aspects where each variation differs, and values for each of those product aspects.
See the Field Index to learn which calls use specifications.
