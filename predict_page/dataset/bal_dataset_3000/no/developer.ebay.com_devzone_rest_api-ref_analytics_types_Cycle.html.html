Cycle

A complex type that describes a program cycle.

Type that uses Cycle:

    StandardsProfile

Calls that use Cycle:

    /seller_standards_profile GET , /seller_standards_profile/{program}/{cycle} GET

Fields

cycleType ( CycleTypeEnum ) [0..?]
The cycle type, either CURRENT or PROJECTED.
See the Field Index to learn which calls use cycleType.

evaluationDate ( string ) [0..?]
The date on which the current, effective seller level was computed.
See the Field Index to learn which calls use evaluationDate.

evaluationMonth ( string ) [0..?]
The month in which the current, effective seller level was computed.
See the Field Index to learn which calls use evaluationMonth.
