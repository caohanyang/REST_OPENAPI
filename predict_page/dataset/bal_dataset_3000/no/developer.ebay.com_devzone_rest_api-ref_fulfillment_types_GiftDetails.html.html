GiftDetails

This type contains information about a line item that was purchased as a gift.

Type that uses GiftDetails:

    LineItem

Calls that use GiftDetails:

    /order GET , /order/{orderId} GET

Fields

message ( string ) [0..?]
A gift message from the buyer to the gift recipient.
See the Field Index to learn which calls use message.

recipientEmail ( string ) [0..?]
The email address of the gift recipient.
See the Field Index to learn which calls use recipientEmail.

senderName ( string ) [0..?]
The name of the buyer to provide to the gift recipient.
See the Field Index to learn which calls use senderName.
