SellingLimit

Defines the selling limit applied to an eBay seller's account.

Type that uses SellingLimit:

    SellingPrivileges

Call that uses SellingLimit:

    /privilege GET

Fields

amount ( Amount ) [0..?]
The maximum amount, in the seller's default currency, that an eBay seller can sell per calendar day.
See the Field Index to learn which calls use amount.

quantity ( integer ) [0..?]
The maximum quantity of items that an eBay seller can sell per calendar day. Note that a listing with variations, all the items listed in the variation count as individual items.
See the Field Index to learn which calls use quantity.
