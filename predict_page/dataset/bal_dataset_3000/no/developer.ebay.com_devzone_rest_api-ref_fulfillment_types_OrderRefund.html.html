OrderRefund

This type contains information about a refund issued for an order. This does not include line item level refunds.

Type that uses OrderRefund:

    PaymentSummary

Calls that use OrderRefund:

    /order GET , /order/{orderId} GET

Fields

amount ( Amount ) [0..?]
The monetary amount of the refund.
See the Field Index to learn which calls use amount.

refundDate ( string ) [0..?]
The date and time that the refund was issued. The timestamp is in ISO 8601 format, which uses the 24-hour Universal Coordinated Time (UTC) clock.

Format: YYYY-MM-DDTHH:MM:SS.SSSZ
Example: 2015-08-04T19:09:02.768Z
See the Field Index to learn which calls use refundDate.

refundReferenceId ( string ) [0..?]
The eBay-generated identifier for the refund
See the Field Index to learn which calls use refundReferenceId.

refundStatus ( RefundStatusEnum ) [0..?]
The degree of completion of a refund being made to the buyer.
See the Field Index to learn which calls use refundStatus.
