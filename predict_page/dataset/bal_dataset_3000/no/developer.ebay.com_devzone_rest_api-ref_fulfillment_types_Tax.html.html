Tax

This type contains information about any tax applied to a line item.

Type that uses Tax:

    LineItem

Calls that use Tax:

    /order GET , /order/{orderId} GET

Field

amount ( Amount ) [0..?]
The monetary amount of the tax.
See the Field Index to learn which calls use amount.
