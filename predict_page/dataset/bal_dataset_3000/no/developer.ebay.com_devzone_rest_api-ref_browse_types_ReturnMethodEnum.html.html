ReturnMethodEnum

Enumeration type that lists the alternative methods to a full refund when an item is returned.

Type that uses ReturnMethodEnum:

    ItemReturnTerms

Call that uses one or more values of ReturnMethodEnum:

    /item/{item_id} GET

Enumeration Values

Value Description
EXCHANGE This value indicates that the buyer has the option of receiving money back for the returned item, or they can exchange the item for another similar item.
REPLACEMENT This value indicates that the buyer has the option of receiving money back for the returned item, or they can choose to have the seller replace the item with an identical item.
