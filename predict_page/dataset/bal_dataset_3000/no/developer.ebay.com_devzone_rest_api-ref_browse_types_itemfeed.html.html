ItemFeed

This type defines the item detail fields returned in the feed file. The file is tab separated and will have the following Header, which indicates the order of the values on each line. The header labels match the fields that are described below.

The following are the header labels in the order in which they are returned.

ItemID Title Description ImageUrl AdditionalImageUrls CategoryId Category Color SizeType Size Material AgeGroup Gender SellerUsername SellerFeedbackScore SellerFeedbackPercentage GTIN Brand MPN ConditionId Condition Quantity PriceValue PriceCurrency ItemEndDate ShipsTo



Type that uses ItemFeed:

    ItemFeedResponse

Call that uses ItemFeed:

    /item_feed GET

additionalImageUrls ( array of string ) [0..?]
An array of item image URLs that does not include the primary image for the item. The URL for the primary image is returned in the imageUrl field.
See the Field Index to learn which calls use additionalImageUrls.

ageGroup ( string ) [0..?]
The age group for which the product is recommended. For example, newborn, infant, toddler, kids, adult, etc.
See the Field Index to learn which calls use ageGroup.

brand ( string ) [0..?]
The name brand of the item, such as Nike, Apple, etc.
See the Field Index to learn which calls use brand.

category ( string ) [0..?]
The name of the eBay category associated with the item.
See the Field Index to learn which calls use category.

categoryId ( string ) [0..?]
The identifier of the leaf category associated with the item.
See the Field Index to learn which calls use categoryId.

color ( string ) [0..?]
The text describing the color of the item.
See the Field Index to learn which calls use color.

condition ( string ) [0..?]
The text describing the condition of the item.
See the Field Index to learn which calls use condition.

conditionId ( string ) [0..?]
The identifier of the condition of the item.
See the Field Index to learn which calls use conditionId.

description ( string ) [0..?]
Text describing the item.
See the Field Index to learn which calls use description.

gender ( string ) [0..?]
The gender for the item. This is used for items that could vary by gender, such as clothing. For example: male, female, or unisex.
See the Field Index to learn which calls use gender.

gtin ( string ) [0..?]
The unique Global Trade Item, (UPC, ISBN, EAN) number of the item as defined by http://www.gtin.info .
See the Field Index to learn which calls use gtin.

imageUrl ( string ) [0..?]
The URL to the primary image of the item. The other images of the item are returned in the additionalImageUrls field.
See the Field Index to learn which calls use imageUrl.

itemEndDate ( datetime ) [0..?]
A timestamp indicating when the item's sale period will end based on its start date and duration. The format is yyyy-mm-dd.
See the Field Index to learn which calls use itemEndDate.

itemId ( string ) [0..?]
The unique identifier of the eBay item.
See the Field Index to learn which calls use itemId.

material ( string ) [0..?]
The text describing what the item is made of. For example, silk or plastic.
See the Field Index to learn which calls use material.

mpn ( string ) [0..?]
The manufacturer part number, which is a unique number that identifies a specific product.
See the Field Index to learn which calls use mpn.

priceCurrency ( CurrencyCodeEnum ) [0..?]
The currency used for the price of the item. Generally, this is the currency used by the country of eBay site offering the item.
See the Field Index to learn which calls use priceCurrency.

priceValue ( string ) [0..?]
The dollar amount of the item specified in the currency shown in the priceCurrency field.
See the Field Index to learn which calls use priceValue.

quantity ( integer ) [0..?]
A number representing how many of this item are available for purchase.
See the Field Index to learn which calls use quantity.

sellerFeedbackPercentage ( string ) [0..?]
The percentage of total feedback that is positive.
See the Field Index to learn which calls use sellerFeedbackPercentage.

sellerFeedbackScore ( integer ) [0..?]
The feedback score of the seller.
See the Field Index to learn which calls use sellerFeedbackScore.

sellerUsername ( string ) [0..?]
The seller's eBay user name.
See the Field Index to learn which calls use sellerUsername.

shipsTo ( string ) [0..?]
The 2 letter ISO 3166-1 alpha-2 code representing the country to which the item can be shipped. Multiple country codes are separated by a , (comma).

See Item Feed API Ships to Locations for information on the countries supported.

See the Field Index to learn which calls use shipsTo.

size ( string ) [0..?]
The size of the item. For example, '7' for a size 7 shoe.
See the Field Index to learn which calls use size.

sizeType ( string ) [0..?]
Text describing a size group in which the item would be included, such as regular, petite, plus, big-and-tall, or maternity.
See the Field Index to learn which calls use sizeType.

title ( string ) [0..?]
The text used as the title of the item.
See the Field Index to learn which calls use title.
