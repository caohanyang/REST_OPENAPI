ShippingOptionSummary

This type is used by the shippingOptions container in the response of the Search for Items call, and consists of shipping cost information.

Type that uses ShippingOptionSummary:

    ItemSummary

Call that uses ShippingOptionSummary:

    /item_summary/search GET

Fields

shippingCost ( Amount_0 ) [0..?]
This is the estimated price to ship the item.
See the Field Index to learn which calls use shippingCost.

shippingCostType ( string ) [0..?]
This field indicates the type of shipping used to ship the item. Possible values are FIXED (flat-rate shipping) and CALCULATED (calculated shipping).
See the Field Index to learn which calls use shippingCostType.
