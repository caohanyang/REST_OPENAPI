CategoryScopeEnum

Enumeration type that indicates the source of the category identifier. This can be either an eBay category ID or a seller's store category ID.

Type that uses CategoryScopeEnum:

    SelectionRule

Calls that use one or more values of CategoryScopeEnum:

    /ad_campaign POST , /ad_campaign/find_campaign_by_ad_reference GET , /ad_campaign/get_campaign_by_name GET , /ad_campaign/{campaign_id} GET

Enumeration Values

Value Description
MARKETPLACE Indicates the identifier is an eBay category ID.
STORE Indicates the identifier is a seller store category ID.
* See the Enumeration Index to see exact use of each enumeration value in the API.
