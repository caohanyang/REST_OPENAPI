PaymentMethodTypeEnum

An enumeration type for the values that indicate the method of payment.

Types that use PaymentMethodTypeEnum:

    PaymentInstrument , PaymentMethod , ProvidedPaymentInstrument

Calls that use one or more values of PaymentMethodTypeEnum:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST , /purchase_order/{purchaseOrderId} GET

Enumeration Value

Value Description
