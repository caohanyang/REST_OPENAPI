StandardsLevelEnum

An enum that defines the eBay seller ratings.

Types that use StandardsLevelEnum:

    Metric , StandardsProfile

Calls that use one or more values of StandardsLevelEnum:

    /seller_standards_profile GET , /seller_standards_profile/{program}/{cycle} GET

Enumeration Values

Value Description
ABOVE_STANDARD Indicates the seller or metric meets the Above Standard requirement.
BELOW_STANDARD Indicates the seller or metric does not meet the minimum standards performance requirement.
TOP_RATED Indicates the seller or metric meets the Top Rated requirement.
* See the Enumeration Index to see exact use of each enumeration value in the API.
