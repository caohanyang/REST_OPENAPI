Region

This type defines information for a region.

Type that uses Region:

    RegionSet

Calls that use Region:

    /fulfillment_policy POST , /fulfillment_policy/get_by_policy_name GET , /fulfillment_policy/{fulfillmentPolicyId} GET

Fields

regionName ( string ) [0..?]
A free-form text string that indicates the name of the region. This value can be the name of a world region (such as the "Middle East" or "Southeast Asia"), a country, or a domestic region within a country (such as "Alaska/Hawaii" or "US Protectorates") depending on the value of regionType . This value should be WORLDWIDE if the regionType value is WORLDWIDE .
See the Field Index to learn which calls use regionName.

regionType ( RegionTypeEnum ) [0..?]
Retrieves the region's type. This value can be 'WORLDWIDE', 'WORLD_REGION', 'COUNTRY', or 'COUNTRY_REGION'.
See the Field Index to learn which calls use regionType.
