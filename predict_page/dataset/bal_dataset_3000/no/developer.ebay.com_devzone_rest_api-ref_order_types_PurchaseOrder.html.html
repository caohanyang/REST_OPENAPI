PurchaseOrder

Type that defines the fields for a purchase order, including line items, costs and charges, payment method, and the purchase order status.

Call that uses PurchaseOrder:

    /purchase_order/{purchaseOrderId} GET

Fields

lineItems ( array of LineItem_0 )
paymentInstrument ( PaymentInstrument )
pricingSummary ( PricingSummary_0 )
purchaseOrderCreationDate ( string )
purchaseOrderId ( string )
purchaseOrderPaymentStatus ( PurchaseOrderPaymentStatusEnum )
purchaseOrderStatus ( PurchaseOrderStatusEnum )
refundedAmount ( Amount_0 )
shippingAddress ( ShippingAddress_0 )
shippingFulfillments ( array of ShippingFulfillment )
warnings ( array of ErrorDetailV3 )


lineItems ( array of LineItem_0 ) [0..?]
Type that defines the fields for the line items in the purchase order.
See the Field Index to learn which calls use lineItems.

paymentInstrument ( PaymentInstrument ) [0..?]
List of payment methods that can be used for the purchase order. Currently only CREDIT_CARD is supported.
See the Field Index to learn which calls use paymentInstrument.

pricingSummary ( PricingSummary_0 ) [0..?]
Container for the details of the grand total of the order.
See the Field Index to learn which calls use pricingSummary.

purchaseOrderCreationDate ( string ) [0..?]
This timestamp indicates the date and time when the purchase order was created.
See the Field Index to learn which calls use purchaseOrderCreationDate.

purchaseOrderId ( string ) [0..?]
The unique identifier of the purchase order. This value was returned in the purchaseOrderId field by the Checkout API's Place Order and Place Guest Order calls.
See the Field Index to learn which calls use purchaseOrderId.

purchaseOrderPaymentStatus ( PurchaseOrderPaymentStatusEnum ) [0..?]
Container for the status of the payment for the purchase order.
See the Field Index to learn which calls use purchaseOrderPaymentStatus.

purchaseOrderStatus ( PurchaseOrderStatusEnum ) [0..?]
Container for the current status of the buyer's payment and any refund that applies to the purchase order.
See the Field Index to learn which calls use purchaseOrderStatus.

refundedAmount ( Amount_0 ) [0..?]
Refund Amount for purchase order
See the Field Index to learn which calls use refundedAmount.

shippingAddress ( ShippingAddress_0 ) [0..?]
The shipping address for the purchase order.
See the Field Index to learn which calls use shippingAddress.

shippingFulfillments ( array of ShippingFulfillment ) [0..?]
An array of the shipping providers and the purchase order delivery details.
See the Field Index to learn which calls use shippingFulfillments.

warnings ( array of ErrorDetailV3 ) [0..?]
An array of warning messages. These type of errors do not prevent the call from executing but should be checked.
See the Field Index to learn which calls use warnings.
