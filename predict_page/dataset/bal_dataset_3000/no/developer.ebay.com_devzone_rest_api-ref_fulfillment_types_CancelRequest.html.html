CancelRequest

This type contains information about a buyer request to cancel an order.

Type that uses CancelRequest:

    CancelStatus

Calls that use CancelRequest:

    /order GET , /order/{orderId} GET

Fields

cancelCompletedDate ( string )
cancelInitiator ( string )
cancelReason ( string )
cancelRequestedDate ( string )
cancelRequestId ( string )
cancelRequestState ( CancelRequestStateEnum )


cancelCompletedDate ( string ) [0..?]
The date and time that the order cancellation was completed, if applicable. The timestamp is in ISO 8601 format, which uses the 24-hour Universal Coordinated Time (UTC) clock.

Format: YYYY-MM-DDTHH:MM:SS.SSSZ
Example: 2015-08-04T19:09:02.768Z
See the Field Index to learn which calls use cancelCompletedDate.

cancelInitiator ( string ) [0..?]
The party who made the initial cancellation request.
See the Field Index to learn which calls use cancelInitiator.

cancelReason ( string ) [0..?]
The reason why the cancelInitiator initiated the cancellation request.
See the Field Index to learn which calls use cancelReason.

cancelRequestedDate ( string ) [0..?]
The date and time that the order cancellation was requested. The timestamp is in ISO 8601 format, which uses the 24-hour Universal Coordinated Time (UTC) clock.

Format: YYYY-MM-DDTHH:MM:SS.SSSZ
Example: 2015-08-04T19:09:02.768Z
See the Field Index to learn which calls use cancelRequestedDate.

cancelRequestId ( string ) [0..?]
The unique identifier of the order cancellation request.
See the Field Index to learn which calls use cancelRequestId.

cancelRequestState ( CancelRequestStateEnum ) [0..?]
The current stage or condition of the cancellation request.
See the Field Index to learn which calls use cancelRequestState.
