TypedNameValue

Type that defines the fields for the name/value pairs for item aspects.

Types that use TypedNameValue:

    Item , ItemDigest , ItemGroup

Calls that use TypedNameValue:

    /item/{item_id} GET , /item_group/{item_group_id} GET

Fields

name ( string ) [0..?]
The text representing the name of the aspect for the name/value pair, such as Color.
See the Field Index to learn which calls use name.

type ( ValueTypeEnum ) [0..?]
Indicates if the value being returned is a string or an array of values.
See the Field Index to learn which calls use type.

value ( any ) [0..?]
The value of the aspect for the name/value pair, such as Red.
See the Field Index to learn which calls use value.
