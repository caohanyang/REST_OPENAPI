UpdateShippingOption

Type that defines the fields needed to update the shipping option of a line item.

Calls that use UpdateShippingOption:

    /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

lineItemId ( string ) [0..?]
A unique eBay-assigned Id value that identifies the line item in a checkout session.
See the Field Index to learn which calls use lineItemId.

shippingOptionId ( string ) [0..?]
A unique Id for the selected shipping option/method.
See the Field Index to learn which calls use shippingOptionId.
