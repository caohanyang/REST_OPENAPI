FilterField

Type that defines result set filters. By default, the items returned are always fixed-price (Buy It Now) and only from Top Rated eBay sellers. These are additional filters you can use on the returned items.

Call that uses FilterField:

    /item_summary/search GET

Fields

field ( string )
negated ( boolean )
range ( RangeValue )
set ( array of string )
value ( string )


field ( string ) [0..?]
Provides a way to filter the results based on a field name and value . The value is separated from the field name with a : (colon). See Search Filters Usage for a list of fields and how to use them to filter the results.

You can also filter by multiple fields. The example below filters the results by the values in the seller and price fields.


&filter;=sellers:{rpsseller}
&filter;=price:[10..50]









See the Field Index to learn which calls use field.

negated ( boolean ) [0..?]
Note: Currently this filter is not supported.
See the Field Index to learn which calls use negated.

range ( RangeValue ) [0..?]
Provides a way to filter the results based on a range of values. The range is specified with a lower value (can be empty, which equates to lesser than or equal to the upper value) and a upper value (can be empty, which equates to greater than or equal to lower value) separated by ... The range values can be inclusive ( [ , ] ) or exclusive ( ( , ) ) or a combination.

For example:
&filter;=price:[10..50] which means price.value is >= 10 and <=50
&filter;=price[..200] which means price.value is <= 200
&filter;=price[200..] or &filter;=price[200] which means price.value is >= 200




See the Field Index to learn which calls use range.

set ( array of string ) [0..?]
Provides a way to filter the results based on a "list of values", where the results has any of the specified values. The values are specified with in { } separated by | (pipe).

For example:
&filter;=sellers:{rpsseller|bigSal|fredsOutdoors} which means seller.userName = rpsseller or bigSal or fredsOutdoors


See the Field Index to learn which calls use set.

value ( string ) [0..?]
Provides a way to filter the results based on a value.

For example:
&filter;=seller:rpsseller which means seller.userName = rpsseller


See the Field Index to learn which calls use value.
