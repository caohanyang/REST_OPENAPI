CancelStatus

This type contains information about any requests that have been made to cancel an order.

Type that uses CancelStatus:

    Order

Calls that use CancelStatus:

    /order GET , /order/{orderId} GET

Fields

cancelledDate ( string ) [0..?]
The date and time the order was cancelled, if applicable. The timestamp is in ISO 8601 format, which uses the 24-hour Universal Coordinated Time (UTC) clock.

Format: YYYY-MM-DDTHH:MM:SS.SSSZ
Example: 2015-08-04T19:09:02.768Z
See the Field Index to learn which calls use cancelledDate.

cancelRequests ( array of CancelRequest ) [0..?]
Contains the details of one or more buyer requests to cancel the order.

For the Get Orders call: This array is returned but is always empty.
For the Get an Order call: This array is returned fully populated with information about any cancellation requests.
See the Field Index to learn which calls use cancelRequests.

cancelState ( CancelStateEnum ) [0..?]
The state of the order with regard to cancellation.
See the Field Index to learn which calls use cancelState.
