ErrorParameterV3

This type is used to indicate the parameter field/value that caused an issue with the call request.

Type that uses ErrorParameterV3:

    ErrorDetailV3

Calls that use ErrorParameterV3:

    /bulk_update_price_quantity POST , /inventory_item/{sku} PUT , /inventory_item/{sku}/product_compatibility PUT , /inventory_item_group/{inventoryItemGroupKey} PUT , /offer POST , /offer/get_listing_fees POST , /offer/{offer_Id} PUT , /offer/{offerId}/publish POST

Fields

name ( string ) [0..?]
This is the name of input field that caused an issue with the call request.
See the Field Index to learn which calls use name.

value ( string ) [0..?]
This is the actual value that was passed in for the element specified in the name field.
See the Field Index to learn which calls use value.
