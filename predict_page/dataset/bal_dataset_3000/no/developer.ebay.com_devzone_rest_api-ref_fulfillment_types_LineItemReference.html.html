LineItemReference

This type identifies a line item in an order.

Types that use LineItemReference:

    ShippingFulfillment , ShippingFulfillmentDetails

Calls that use LineItemReference:

    /order/{orderId}/shipping_fulfillment POST , /order/{orderId}/shipping_fulfillment/{fulfillmentId} GET

Fields

lineItemId ( string ) [0..?]
The globally unique eBay-generated identifier of the line item.

Note: A single line item can consist of multiple units of a purchased item, and one unit can consist of multiple parts or components. Depending on how those components are provided by the manufacturer, the seller may be able to distribute the components among multiple packages in a variety of ways. Thus, a given line item ID can appear in multiple fulfillments.
See the Field Index to learn which calls use lineItemId.

quantity ( integer ) [0..?]
This field is reserved for internal or future use.
See the Field Index to learn which calls use quantity.
