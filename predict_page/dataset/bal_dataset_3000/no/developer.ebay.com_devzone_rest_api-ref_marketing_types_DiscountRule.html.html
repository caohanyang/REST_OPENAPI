DiscountRule

Type that defines the fields for the discount percentage or amount of the promotion. Refer to Item Promotion Rules for more information about using the discountRule and discountSpecificiation fields together to create various promotions.

Required unless the promotion is in DRAFT state.

Types that use DiscountRule:

    ItemPromotion , ItemPromotionResponse

Calls that use DiscountRule:

    /item_promotion POST , /item_promotion/{promotion_id} GET

Fields

discountBenefit ( DiscountBenefit ) [0..?]
Container for the details of the buyer's discount.
See the Field Index to learn which calls use discountBenefit.

discountSpecification ( DiscountSpecification ) [0..?]
Container for the quantities and minimum and maximum rules of the discount. Refer to Item Promotion Rules for more information about using the discountRule and discountSpecificiation fields together to create various promotions.

Required unless the promotion is in DRAFT state.

See the Field Index to learn which calls use discountSpecification.

ruleOrder ( integer ) [0..?]
Specifies the order in which to evaluate rules where inventory is share and each has its own set of rules.

Required unless the promotion is in DRAFT state.

See the Field Index to learn which calls use ruleOrder.
