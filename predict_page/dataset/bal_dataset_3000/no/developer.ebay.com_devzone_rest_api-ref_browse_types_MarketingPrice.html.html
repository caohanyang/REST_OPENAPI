MarketingPrice

Type that defines the fields that describe a discount.

Type that uses MarketingPrice:

    Item

Call that uses MarketingPrice:

    /item/{item_id} GET

Fields

discountAmount ( Amount ) [0..?]
This container shows the monetary amount of the discount. This container is returned when a discount applies to the item.
See the Field Index to learn which calls use discountAmount.

discountPercentage ( string ) [0..?]
This field expresses the percentage of the discount based on the value in the originalPrice container.
See the Field Index to learn which calls use discountPercentage.

originalPrice ( Amount ) [0..?]
This container shows the monetary amount of the item.
See the Field Index to learn which calls use originalPrice.
