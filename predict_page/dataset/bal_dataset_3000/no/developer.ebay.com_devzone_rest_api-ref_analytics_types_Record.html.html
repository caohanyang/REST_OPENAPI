Record

Type that defines the fields of the individual record of the report.

Type that uses Record:

    Report

Call that uses Record:

    /traffic_report GET

Fields

dimensionValues ( array of Value ) [0..?]
The container for the dimension value.
See the Field Index to learn which calls use dimensionValues.

metricValues ( array of Value ) [0..?]
The container for the metric value and the indicator of whether the value is valid of one record in the report.
See the Field Index to learn which calls use metricValues.
