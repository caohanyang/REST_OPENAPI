ShippingOption

Type that defines the fields for the details of a shipping provider.

Type that uses ShippingOption:

    Item

Call that uses ShippingOption:

    /item/{item_id} GET

Fields

shippingCarrierName ( string )
shippingCost ( Amount )
shippingServiceName ( string )
trademarkSymbol ( string )
type ( string )


shippingCarrierName ( string ) [0..?]
A name of the shipping provider, such as FedEx, or USPS.
See the Field Index to learn which calls use shippingCarrierName.

shippingCost ( Amount ) [0..?]
The final shipping cost for all the items after all discounts are applied.
See the Field Index to learn which calls use shippingCost.

shippingServiceName ( string ) [0..?]
A name of a shipping type. For example, Priority Mail Express (provided by USPS) or FedEx International Priority (Provided by FedEx).
See the Field Index to learn which calls use shippingServiceName.

trademarkSymbol ( string ) [0..?]
Any trademark symbol, such as ™ or ®, that needs to be shown in superscript next to the shipping service name.
See the Field Index to learn which calls use trademarkSymbol.

type ( string ) [0..?]
The type of a shipping option, such as EXPEDITED, ONE_DAY, STANDARD, ECONOMY, PICKUP, etc.
See the Field Index to learn which calls use type.
