GeoCoordinates

This type is used to express the Global Positioning System (GPS) latitude and longitude coordinates of an inventory location.

Type that uses GeoCoordinates:

    Location

Calls that use GeoCoordinates:

    /location GET , /location/{merchantLocationKey} POST

Fields

latitude ( number ) [0..?]
Required if geoCoordinates are used. The latitude (North-South) component of the geographic coordinate.

Returned if geographical coordinates are set for the inventory location.
See the Field Index to learn which calls use latitude.

longitude ( number ) [0..?]
Required if geoCoordinates are used. The longitude (East-West) component of the geographic coordinate.

Returned if geographical coordinates are set for the inventory location.
See the Field Index to learn which calls use longitude.
