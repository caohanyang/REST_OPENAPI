ItemDigest

Type that defines the fields for the details of an item, such as brand, price, title, etc.

Type that uses ItemDigest:

    ItemGroup

Call that uses ItemDigest:

    /item_group/{item_group_id} GET

Fields

availabilityStatusForShipToHome ( AvailabilityStatusEnum )

ageGroup ( string ) [0..?]
(Item Aspect) The age group for which the product is recommended. For example, newborn, infant, toddler, kids, adult, etc. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use ageGroup.

availabilityStatusForShipToHome ( AvailabilityStatusEnum ) [0..?]
The availability of this item.
See the Field Index to learn which calls use availabilityStatusForShipToHome.

brand ( string ) [0..?]
(Item Aspect) The name brand of the item, such as Nike, Apple, etc. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use brand.

color ( string ) [0..?]
(Item Aspect) The text describing the color of the item. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use color.

gender ( string ) [0..?]
(Item Aspect) The gender for the item. This is used for items that could vary by gender, such as clothing. For example: male, female, or unisex. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use gender.

image ( Image ) [0..?]
The URL to the primary image of the item. The other images of the item are returned in the additionalImages field.
See the Field Index to learn which calls use image.

itemHref ( string ) [0..?]
A HATEOAS reference you use to retrieve an item.

For example:
/buy/v1/item/v1|150006693008|0


See the Field Index to learn which calls use itemHref.

itemId ( string ) [0..?]
The unique identifier of an item.
See the Field Index to learn which calls use itemId.

localizedAspects ( array of TypedNameValue ) [0..?]
(Additional Item Aspects) A list of name/value pairs that describe additional item aspects. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use localizedAspects.

material ( string ) [0..?]
(Item Aspect) The text describing what the item is made of. For example, silk. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use material.

pattern ( string ) [0..?]
(Item Aspect) The text describing the pattern used on the item. For example, paisley. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use pattern.

price ( Amount ) [0..?]
The cost of the item.
See the Field Index to learn which calls use price.

priceDisplayCondition ( PriceDisplayConditionEnum ) [0..?]
This enumeration value indicates how item price is displayed. For some items, the prices are so low that they can only be shown to the buyer after they add the item to the cart.
See the Field Index to learn which calls use priceDisplayCondition.

quantitySold ( integer ) [0..?]
The number of items that have been sold. This field is only applicable to multi-quantity, fixed-price listings.
See the Field Index to learn which calls use quantitySold.

size ( string ) [0..?]
(Item Aspect) The size of the item. For example, '7' for a size 7 shoe. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use size.

sizeSystem ( string ) [0..?]
(Item Aspect) The sizing system of the country. Other aspects are returned in the localizedAspects container.

Valid Values:
US
UK
EU
DE
FR
JP
CN (China)
IT
BR
MEX
AU












See the Field Index to learn which calls use sizeSystem.

sizeType ( string ) [0..?]
(Item Aspect) The text describing a size group in which the item would be included, such as regular, petite, plus, big-and-tall, maternity. Other aspects are returned in the localizedAspects container.
See the Field Index to learn which calls use sizeType.

title ( string ) [0..?]
The seller-created title of the item.
See the Field Index to learn which calls use title.
