ReviewRating

Type that defines the information for the rating of a product review.

Types that use ReviewRating:

    Item , ItemGroup

Calls that use ReviewRating:

    /item/{item_id} GET , /item_group/{item_group_id} GET

Fields

averageRating ( string ) [0..?]
The average rating given to a product based on customer reviews.
See the Field Index to learn which calls use averageRating.

ratingHistograms ( array of RatingHistogram ) [0..?]
The product rating histogram that shows the review counts and the product rating.
See the Field Index to learn which calls use ratingHistograms.

reviewCount ( integer ) [0..?]
The total number of reviews for the item.
See the Field Index to learn which calls use reviewCount.
