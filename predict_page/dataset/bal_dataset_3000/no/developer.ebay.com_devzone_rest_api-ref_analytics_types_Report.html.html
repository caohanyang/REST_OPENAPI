Report

Type that defines the fields of the report.

Call that uses Report:

    /traffic_report GET

Fields

endDate ( string )
header ( Header )
lastUpdatedDate ( string )
records ( array of Record )
startDate ( string )
warnings ( array of ErrorDetailV3 )


endDate ( string ) [0..?]
The end date range for the data in this report.

If you specify an end date that is beyond the lastUpdatedDate , eBay returns a report that contains data only up to the lastUpdatedDate.

See the Field Index to learn which calls use endDate.

header ( Header ) [0..?]
Container for the header, which is specified in the request in the dimension field.
See the Field Index to learn which calls use header.

lastUpdatedDate ( string ) [0..?]
The date and time the report was last updated.
See the Field Index to learn which calls use lastUpdatedDate.

records ( array of Record ) [0..?]
Container for the individual records.
See the Field Index to learn which calls use records.

startDate ( string ) [0..?]
The start of the date range that the report covers.
See the Field Index to learn which calls use startDate.

warnings ( array of ErrorDetailV3 ) [0..?]
Warnings. Applicable only when you specify a set of listing IDs
See the Field Index to learn which calls use warnings.
