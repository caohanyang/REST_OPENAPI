DeliveryCost

This type contains a breakdown of all costs associated with the fulfillment of a line item.

Type that uses DeliveryCost:

    LineItem

Calls that use DeliveryCost:

    /order GET , /order/{orderId} GET

Fields

importCharges ( Amount ) [0..?]
The amount of any import charges applied to international shipping of the line item.
See the Field Index to learn which calls use importCharges.

shippingCost ( Amount ) [0..?]
The total cost of shipping all units of the line item.
See the Field Index to learn which calls use shippingCost.

shippingIntermediationFee ( Amount ) [0..?]
The amount of any fee charged by a provider of shipping intermediation for the line item.
See the Field Index to learn which calls use shippingIntermediationFee.
