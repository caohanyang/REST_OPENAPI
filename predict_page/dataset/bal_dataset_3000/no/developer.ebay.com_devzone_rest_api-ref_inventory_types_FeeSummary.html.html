FeeSummary

This type is used to display the expected listing fees for each unpublished offer specified in the request of the Get Listing Fees call.

Type that uses FeeSummary:

    FeesSummaryResponse

Call that uses FeeSummary:

    /offer/get_listing_fees POST

Fields

fees ( array of Fee ) [0..?]
This container is an array of listing fees that can be expected to be applied to an offer on the specified eBay marketplace ( marketplaceId value). Many fee types will get returned even when they are 0.0 .
See the Field Index to learn which calls use fees.

marketplaceId ( MarketplaceEnum ) [0..?]
This is the unique identifier of the eBay site for which listing fees for the offer are applicable.
See the Field Index to learn which calls use marketplaceId.

warnings ( array of ErrorDetailV3 ) [0..?]
This container will contain an array of errors and/or warnings when a call is made, and errors and/or warnings occur.
See the Field Index to learn which calls use warnings.
