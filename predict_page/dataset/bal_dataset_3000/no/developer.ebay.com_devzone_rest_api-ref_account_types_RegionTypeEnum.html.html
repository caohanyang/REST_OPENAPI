RegionTypeEnum

This enumerated type defines region types.

Type that uses RegionTypeEnum:

    Region

Calls that use one or more values of RegionTypeEnum:

    /fulfillment_policy POST , /fulfillment_policy/get_by_policy_name GET , /fulfillment_policy/{fulfillmentPolicyId} GET

Enumeration Values

Value Description
COUNTRY A single country.
COUNTRY_REGION A country, or a domestic region within a country.
STATE_OR_PROVINCE A state or province, such as Alaska or Hawaii, or the US Protectorates.
WORLD_REGION A world region, such as the Middle East or Southeast Asia.
WORLDWIDE The entire world as we know it.
* See the Enumeration Index to see exact use of each enumeration value in the API.
