ShippingOption

Type that defines the fields for the shipping options.

Type that uses ShippingOption:

    LineItem

Calls that use ShippingOption:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

baseDeliveryCost ( Amount )
deliveryDiscount ( Amount )
maxEstimatedDeliveryDate ( string )

baseDeliveryCost ( Amount ) [0..?]
The shipping cost using this shipping option for this line item before any shipping discounts.
See the Field Index to learn which calls use baseDeliveryCost.

deliveryDiscount ( Amount ) [0..?]
The delivery discount
See the Field Index to learn which calls use deliveryDiscount.

maxEstimatedDeliveryDate ( string ) [0..?]
The end of the date range in which the purchase order is expected to be delivered to the shipping address.
See the Field Index to learn which calls use maxEstimatedDeliveryDate.

minEstimatedDeliveryDate ( string ) [0..?]
The beginning of the date range in which the purchase order is expected to be delivered to the shipping address.
See the Field Index to learn which calls use minEstimatedDeliveryDate.

selected ( boolean ) [0..?]
Indicates if the shipping method is selected.
See the Field Index to learn which calls use selected.

shippingCarrierName ( string ) [0..?]
A name of the shipping provider, such as FedEx, or USPS.
See the Field Index to learn which calls use shippingCarrierName.

shippingOptionId ( string ) [0..?]
A unique Id for the selected shipping option/method.
See the Field Index to learn which calls use shippingOptionId.

shippingServiceName ( string ) [0..?]
A name of a shipping type. For example, Priority Mail Express (provided by USPS) or FedEx International Priority (Provided by FedEx).
See the Field Index to learn which calls use shippingServiceName.
