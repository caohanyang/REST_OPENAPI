Category

This type is used by the categories container in the response of the Search for Items call, and consists of the primary listing category ID of the item (as well as secondary listing category if item was listed in two categories).

Type that uses Category:

    ItemSummary

Call that uses Category:

    /item_summary/search GET

Field

categoryId ( string ) [0..?]
The unique identifier of the primary listing category of the item (as well as secondary listing category if item was listed in two categories).
See the Field Index to learn which calls use categoryId.
