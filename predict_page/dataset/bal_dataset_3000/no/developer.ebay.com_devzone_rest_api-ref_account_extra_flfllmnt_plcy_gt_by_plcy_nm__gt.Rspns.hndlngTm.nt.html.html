Applicable values for /fulfillment_policy/get_by_policy_name GETResponse.handlingTime.unit

BUSINESS_DAY
Time duration is based on the number of business days, meaning 'working days' (normally Monday through Friday). This excludes all holidays in the time duration, and depending on the location, it can include or exclude Saturday.
CALENDAR_DAY
Time duration is based on the calendar week, meaning Sunday through Saturday. This time duration does not exclude holidays.
DAY
Time duration is based on the number of days.
HOUR
Time duration is based on the number of hours.
MILLISECOND
Time duration is based on the number of milliseconds.
MINUTE
Time duration is based on the number of minutes.
MONTH
Time duration is based on the number of months
SECOND
Time duration is based on the number of seconds.
YEAR
Time duration is based on the number of years.
