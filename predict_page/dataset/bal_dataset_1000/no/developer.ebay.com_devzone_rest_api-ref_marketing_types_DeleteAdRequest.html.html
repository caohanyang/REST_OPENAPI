DeleteAdRequest

The type that defines the fields for the delete ad request.

Type that uses DeleteAdRequest:

    BulkDeleteAdRequest

Call that uses DeleteAdRequest:

    /ad_campaign/{campaign_id}/bulk_delete_ads_by_listing_id POST

Field

listingId ( string ) [0..?]
The eBay listing identifier. This ID was generated when the listing was created.
See the Field Index to learn which calls use listingId.
