Interval

This type is used by the intervals container to define the opening and closing times of a store's working day. Local time (in Military format) is used, with the following format: hh:mm:ss .

Note: This type is not yet applicable since establishing In-Store Pickup or Click and Collect inventory is not supported with the the initial release of the Inventory API.

Types that use Interval:

    OperatingHours , SpecialHours

Calls that use Interval:

    /location GET , /location/{merchantLocationKey} POST , /location/{merchantLocationKey}/update_location_details POST

Fields

close ( string ) [0..?]
Required if the intervals container is used to specify working hours or special hours for a store. The close value is actually the time that the store closes. Local time (in Military format) is used. So, if a store closed at 8 PM local time, the close time would look like the following: 20:00:00 .

Returned if set for the store location.
See the Field Index to learn which calls use close.

open ( string ) [0..?]
Required if the intervals container is used to specify working hours or special hours for a store. The open value is actually the time that the store opens. Local time (in Military format) is used. So, if a store opened at 9 AM local time, the open time would look like the following: 09:00:00 .

Returned if set for the store location.
See the Field Index to learn which calls use open.
