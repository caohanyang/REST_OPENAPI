Weight

This type is used to specify the weight (and the unit used to measure that weight) of a shipping package.

Type that uses Weight:

    PackageWeightAndSize

Calls that use Weight:

    /inventory_item GET , /inventory_item/{sku} PUT

Fields

unit ( WeightUnitOfMeasureEnum ) [0..?]
Required if weight container is used. The unit of measurement used to specify the weight of a shipping package. If the English system of measurement is being used, the applicable values for weight units are POUND and OUNCE . If the metric system of measurement is being used, the applicable values for weight units are KILOGRAM and GRAM . The metric system is used by most countries outside of the US.
See the Field Index to learn which calls use unit.

value ( number ) [0..?]
Required if weight container is used. The actual weight (in the measurement unit specified in the unit field) of the shipping package. If a shipping package weighed 20.5 ounces, the container would look as follows: 
"weight": {
 "value": 20.5,
 "unit": "OUNCE"
 }
See the Field Index to learn which calls use value.
