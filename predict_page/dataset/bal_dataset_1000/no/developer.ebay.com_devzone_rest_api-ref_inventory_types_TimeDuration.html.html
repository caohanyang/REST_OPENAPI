TimeDuration

This type is used by the fulfillmentTime field of the PickupAtLocationAvailability type to specify how soon the item will be ready for In-Store pickup or Click and Collect after the order has occurred.

Note: The TimeDuration type is currently not applicable, since this type is only used by the PickupAtLocationAvailability type, which is not yet available for use. In the future, the PickupAtLocationAvailability type will allow sellers to set the quantity of the inventory item that are available for In-Store Pickup or Click and Collect at one or more of the merchant's physical stores.

Type that uses TimeDuration:

    PickupAtLocationAvailability

Calls that use TimeDuration:

    /inventory_item GET , /inventory_item/{sku} PUT

Fields

unit ( TimeDurationUnitEnum ) [0..?]
This enumeration value indicates the time unit used to specify the fulfillment time, such as HOUR .

Note: The fulfillmentTime container and its child fields are not yet available for use.
See the Field Index to learn which calls use unit.

value ( integer ) [0..?]
The integer value in this field, along with the time unit in the unit field will indicate how soon after the order takes place that the buyer can pick up the item at the store. If the value of this field is 4 , and the value of the unit field is HOUR , then the fulfillment time is 4 hours, which means that the buyer will be able to pick up the item at the store four hours after the transaction took place.

Note: The fulfillmentTime container and its child fields are not yet available for use.
See the Field Index to learn which calls use value.
