LookbackPeriod

Defines a span of time in the past.

No calls use LookbackPeriod.

Fields

lifetime ( boolean ) [0..?]
Not used by any call.

When set to true , the lookback period is the lifetime of the account (the start time being when the account was opened to the current time).
See the Field Index to learn which calls use lifetime.

timeDuration ( TimeDuration ) [0..?]
Not used by any call.

The duration of the lookback period, in the units defined by TimeDurationUnitEnum .
See the Field Index to learn which calls use timeDuration.
