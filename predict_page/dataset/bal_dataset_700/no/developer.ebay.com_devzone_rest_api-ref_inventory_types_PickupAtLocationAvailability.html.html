PickupAtLocationAvailability

This type is used to specify/indicate the quantity of the inventory item that is available for an In-Store Pickup or Click and Collect order at the merchant's physical store (specified by the merchantLocationKey field).

Note: The PickupAtLocationAvailability type is not yet available for use. In the future, this type will allow sellers to set the quantity of the inventory item that are available for In-Store Pickup or Click and Collect at one or more of the merchant's physical stores.

Type that uses PickupAtLocationAvailability:

    Availability

Calls that use PickupAtLocationAvailability:

    /inventory_item GET , /inventory_item/{sku} PUT

Fields

availabilityType ( AvailabilityTypeEnum ) [0..?]
Required if the pickupAtLocationAvailability container is used, and is always returned with the pickupAtLocationAvailability container. The enumeration value in this field indicates the availability status of the inventory item at the merchant's physical store specified by the pickupAtLocationAvailability.merchantLocationKey field.

See AvailabilityTypeEnum for more information about how/when you use each enumeration value.

Note: The pickupAtLocationAvailability container and its child fields are not yet available for use.
See the Field Index to learn which calls use availabilityType.

fulfillmentTime ( TimeDuration ) [0..?]
Required if the pickupAtLocationAvailability container is used, and is always returned with the pickupAtLocationAvailability container. This container is used to indicate how soon an In-Store Pickup or Click and Collect order will be available for pickup by the buyer after the order takes place.

Note: The pickupAtLocationAvailability container and its child fields are not yet available for use.
See the Field Index to learn which calls use fulfillmentTime.

merchantLocationKey ( string ) [0..?]
Required if the pickupAtLocationAvailability container is used, and is always returned with the pickupAtLocationAvailability container. The unique identifier of a merchant's store where the In-Store Pickup or Click and Collect inventory item is currently located, or where inventory will be sent to. If the merchant's store is currently awaiting for inventory, the availabilityType value should be SHIP_TO_STORE .

Note: The pickupAtLocationAvailability container and its child fields are not yet available for use.
Max length : 36
See the Field Index to learn which calls use merchantLocationKey.

quantity ( integer ) [0..?]
Required if the pickupAtLocationAvailability container is used, and is always returned with the pickupAtLocationAvailability container. This integer value indicates the quantity of the inventory item that is available for In-Store Pickup at the store identified by the merchantLocationKey value. The value of quantity should be an integer value greater than 0 , unless the inventory item is out of stock.

Note: The pickupAtLocationAvailability container and its child fields are not yet available for use.
See the Field Index to learn which calls use quantity.
