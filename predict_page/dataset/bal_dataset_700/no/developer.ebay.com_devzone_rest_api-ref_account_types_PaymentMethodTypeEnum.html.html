PaymentMethodTypeEnum

An enum type that defines the supported payment methods.

Type that uses PaymentMethodTypeEnum:

    PaymentMethod

Calls that use one or more values of PaymentMethodTypeEnum:

    /payment_policy POST , /payment_policy/get_by_policy_name GET , /payment_policy/{payment_policy_id} PUT , /payment_policy/{paymentPolicyId} GET
