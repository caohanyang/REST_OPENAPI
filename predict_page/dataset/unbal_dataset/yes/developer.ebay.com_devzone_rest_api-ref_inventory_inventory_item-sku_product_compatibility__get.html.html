Get Product Compatibility

GET /inventory_item/{sku}/product_compatibility

This call is used by the seller to retrieve the list of products that are compatible with the inventory item. The inventory item to retrieve is identified with a SKU value in the URI. Product compatibility is currently only applicable to motor vehicle parts and accessory categories, but more categories may be supported in the future.

compatibleProducts. notes string Conditionally This field is optionally used by the seller to input any notes pertaining to the parts compatibility being specified. The seller might use this field to specify the placement of the part on a vehicle or other applicable information. This field will only be returned if specified by the seller.

Either the productIdentifier or the productFamilyProperties container (or both) can be used to specify one or more motor vehicles that are compatible with the inventory item. If the productIdentifier container is used, the product identifiers that are passed in that container must match vehicles in the eBay catalog in order for that vehicle to be added to the compatible vehicle list.

compatibleProducts
.productIdentifier. epid 		string Conditionally This field can be used if the seller already knows the eBay product ID associated with the motor vehicle that is to be added to the compatible product list. If this eBay product ID is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.
compatibleProducts
.productIdentifier. gtin 		string Conditionally This field can be used if the seller knows the Global Trade Identification Number for the motor vehicle that is to be added to the compatible product list. If this GTIN value is found in the eBay product catalog, the motor vehicle properties (engine, make, model, trim, and year) will automatically get picked up for that motor vehicle.
compatibleProducts
.productIdentifier. ktype 		string Conditionally This field can be used if the seller knows the K Type Number for the motor vehicle that is to be added to the compatible product list. Sample: Retrieving a Product Compatibility List

This call retrieves a product compatibility list.

Description

This call will retrieve a product compatibility list that is associated with a particular SKU.

Input

The SKU value of the product whose product compatibility list will be retrieved is input into the end of the call URI. In this case, the SKU value for the product is Al-8730 . 

URL format. See also the non-wrapped version of this URL.

GET https://api.ebay.com/sell/inventory/v1/inventory_item/Al-8730/product_compatibility

Output

Three compatible vehicles are returned for the product, 'Al-8730'. The attributes are given for each of these three vehicles. The SKU value is also returned in the response.
