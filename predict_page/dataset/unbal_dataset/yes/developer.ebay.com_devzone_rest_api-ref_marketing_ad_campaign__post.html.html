Create Campaign

POST /ad_campaign

REQUIREMENTS: To use this API, the seller must have an eBay Store and has accepted the Promoted Listings Terms and Conditions for the site they are using. Below are links for the Terms and Conditions for each site.

AU : http://scgi.ebay.com.au/ws/eBayISAPI.dll?UserAgreementV2&agrid;=17
DE : http://scgi.ebay.de/ws/eBayISAPI.dll?UserAgreementV2&agrid;=17
UK : http://scgi.ebay.co.uk/ws/eBayISAPI.dll?UserAgreementV2&agrid;=17
US : http://scgi.ebay.com/ws/eBayISAPI.dll?UserAgreementV2&agrid;=17




This call creates a campaign. The basic information of a campaign is a name, start and end date, the eBay marketplace that will host the campaign and the funding strategy. The funding strategy includes the funding model that specifies the percentage of the sale price that the seller will be charged by eBay when the listing sells via a promoted listing. The sale of an item via a promoted listing is the event that triggers the seller fee.

In addition to the basic campaign information, a campaign has ads, which are the promoted listings. You can specify the listings by passing in a list of Ids or a list of rules in which to select the listings. Each listing is associated with an ad and the ad is associated with the campaign. Note: A campaign is either a key based (based on a list of Ids) campaign, or a rule based campaign.

  • Create the ads for the campaign with a list of Ids:
    1. Submit the Create Campaign call with the basic campaign information.
    2. Pass in the list of seller inventory reference Ids or a list of eBay listing Ids using one of the Create Ads in Bulk calls, such as, Create Adsin Bulk By Inventory Reference or Create Ads in Bulk By Listing Id .

      You can specify up to 500 listings per call, which results in 500 ads. And each campaign can have a total of 25,000 ads.

  • Create a campaign based on selection rules:

    Submit the Create Campaign call with the basic campaign information and the selection rules.

    When you submit a Create Campaign call containing rules, the service checks the number of listings that comply to the rules. If that number is <= 25,000, the call is successful and the service starts creating the ads.

    Note: If during the time the service is creating the ads from the initial call, the seller adds inventory that complies to the rules, these listing will be included in the campaign up to a total of 25,000 listings. If there are more than 25,000 listing, only the first 25,000 listings are added to the campaign.





On the start date, the service derives the keywords for each listing in the campaign, it creates an ad for each listing and associates that ad with the campaign, and then starts the campaign. The amount of time it takes the service to start the campaign depends on the number of listings in the campaign. You can use the Get Campaign By Name call to check if the processing has finished and the campaign has been created.

Note: The campaign Id is returned in the response Location header.

INVENTORY_PARTITION

  • Each set of selection rules are ORed with each other.
  • Individual rules within a selection rule set are ANDed with each other. If a rule has a list of values, such a list of category IDs, the item must match only one of the values of the rule.
    • Note: If an item matches multiple sets of rules or multiple rules within a selection rule set, the item is considered only once.
Maximum number: 10

fundingStrategy. bidPercentage string Required The default bid percentage for the funding model. This is the percentage of the sale price that the seller will be charged by eBay when the listing sells. If bid percentage is not provided, the default bid percentage associated with the campaign is used. The higher the bid, the better the position of the listing on the page.

The bid percentage is a single precision value. For example, 4.1, 4.10, 5.5, 5.0, etc. are valid values and 4.44, 8.11, etc. would be invalid values.

Minimum value: 1.0

Sample: Create a Campaign for an Inventory Campaign

This sample creates the campaign without items. After the campaign is created, can use the bulkCreateAdsBy calls to add the items and create the ads for the campaign.

Description

This sample creates a campaign named Fall Sale that sets the start date and bid percentage, which will be used for all ads in the campaign.

Input

The inputs for this sample are campaignName , startDate , fundingStrategy.bidPercentage , fundingStrategy.fundingModel and marketplaceId .

   "campaignName":"Fall Sale",
   "startDate":"2016-09-07T21:43:00Z",
   "fundingStrategy":{
      "bidPercentage":"10.0",
      "fundingModel":"COST_PER_SALE"
   },
   "marketplaceId":"EBAY_US"
}
