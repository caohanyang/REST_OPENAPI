Get a Sales Tax Table

GET /sales_tax

Use this call to retrieve a sales tax table that the seller established for a specific country. Specify the tax table to retrieve using the country_code query parameter.

Input Output Error Codes Samples Change History

Input

See also Samples .

Resource URI (production) 

GET https://api.ebay.com/sell/account/v1/sales_tax?
  country_code=CountryCodeEnum

URI parameters

Parameter Type Required? Meaning
country_code string Required A two-letter ISO 3166-1 Alpha-2 .

Applicable values are from CountryCodeEnum .

Sample: Retrieve the sales tax tables for a specified country.

This call gets the sales tax tables for a specified country.

Description

This call gets all the sales tax tables that the seller has configured for the specified country's jurisdictions.

Input

While this call has no payload, you must supply the two-letter country code for which you want to retrieve sales tax tables in the country_code query parameter. 

URL format. See also the non-wrapped version of this URL.

GET https://api.sandbox.ebay.com/sell/account/v1/sales_tax?
   country_code=US

Output

This example call returns the sales tax tables that have been set up for the United States (US). 

JSON format.
{
  "salesTaxes": [
    {
      "salesTaxPercentage": "25.5",
      "shippingAndHandlingTaxed": false,
      "countryCode": "US",
      "salesTaxJurisdictionId": "IL"
    },
    {
      "salesTaxPercentage": "4.4",
      "shippingAndHandlingTaxed": true,
      "countryCode": "US",
      "salesTaxJurisdictionId": "NY"
    }
  ]
}
