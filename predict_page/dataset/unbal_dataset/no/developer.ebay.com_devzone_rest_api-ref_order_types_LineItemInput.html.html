LineItemInput

Type that defines the fields for a line item.

Types that use LineItemInput:

    CreateGuestCheckoutSessionRequest , CreateSignInCheckoutSessionRequest

Calls that use LineItemInput:

    /checkout_session/initiate POST , /guest_checkout_session/initiate POST

Fields

itemId ( string ) [0..?]
The unique eBay-assigned Id for the item. Each itemId will become a single line item. You can have a maximum of four itemId (s) per checkout.

Maximum : 4

See the Field Index to learn which calls use itemId.

quantity ( integer ) [0..?]
The number of items ordered in this line item.
See the Field Index to learn which calls use quantity.
