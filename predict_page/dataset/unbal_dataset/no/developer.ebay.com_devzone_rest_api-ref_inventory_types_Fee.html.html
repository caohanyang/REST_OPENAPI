Fee

This type is used to express expected listing fees for offers, as well as any eBay-related promotional discounts being applied toward a specific fee.

Type that uses Fee:

    FeeSummary

Call that uses Fee:

    /offer/get_listing_fees POST

Fields

amount ( Amount ) [0..?]
This is the actual dollar value of the fee type specified in the feeType field.
See the Field Index to learn which calls use amount.

feeType ( string ) [0..?]
The value returned in this field indicates the type of fee that the seller incurred for the offer that was specified through the call request. Applicable fees will often include things such as InsertionFee or SubtitleFee , but many fee types will get returned even when they are 0.0 .
See the Field Index to learn which calls use feeType.

promotionalDiscount ( Amount ) [0..?]
The dollar value in this field indicates any eBay promotional discount applied toward the listing fee. If there was no discount applied toward the fee, this field is still returned but its value is 0 .
See the Field Index to learn which calls use promotionalDiscount.
