ShippingFulfillment

This type contains the complete details of an existing fulfillment for an order.

Type that uses ShippingFulfillment:

    ShippingFulfillmentPagedCollection

Calls that use ShippingFulfillment:

    /order/{orderId}/shipping_fulfillment GET , /order/{orderId}/shipping_fulfillment/{fulfillmentId} GET

Fields

fulfillmentId ( string )
lineItems ( array of LineItemReference )
shipmentTrackingNumber ( string )
shippedDate ( string )
shippingCarrierCode ( string )


fulfillmentId ( string ) [0..?]
The unique identifier of the fulfillment; for example, 9405509699937003457459 . This eBay-generated value was created by the Create Shipping Fulfillment call.
See the Field Index to learn which calls use fulfillmentId.

lineItems ( array of LineItemReference ) [0..?]
Contains a list of the line items to which the fulfillment applies.
See the Field Index to learn which calls use lineItems.

shipmentTrackingNumber ( string ) [0..?]
The tracking number provided by the shipping carrier for this fulfillment.
See the Field Index to learn which calls use shipmentTrackingNumber.

shippedDate ( string ) [0..?]
The date and time that the fulfillment package was shipped. The timestamp is in ISO 8601 format, which uses the 24-hour Universal Coordinated Time (UTC) clock.

Format: YYYY-MM-DDTHH:MM:SS.SSSZ
Example: 2015-08-04T19:09:02.768Z
See the Field Index to learn which calls use shippedDate.

shippingCarrierCode ( string ) [0..?]
The eBay code identifying the shipping carrier for this fulfillment.

Note: The Trading API's ShippingCarrierCodeType enumeration contains the most current list of eBay shipping carrier codes and the countries served by each carrier. See ShippingCarrierCodeType .
See the Field Index to learn which calls use shippingCarrierCode.
