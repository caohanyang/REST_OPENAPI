CountryCodeEnum

This enumerated type lists the two-letter ISO 3166-1 Alpha-2 code representing a country.

Type that uses CountryCodeEnum:

    SalesTax

Calls that use one or more values of CountryCodeEnum:

    /sales_tax GET , /sales_tax/{countryCode}/{jurisdictionId} GET
