Definition

Type the defines the fields that describe the dimension keys and metrics.

Type that uses Definition:

    Header

Call that uses Definition:

    /traffic_report GET

Fields

dataType ( DataTypeEnum ) [0..?]
Indicates the dimension data type. For example, if the dimension is day , this value will be DATE .
See the Field Index to learn which calls use dataType.

key ( string ) [0..?]
The value submitted in the dimension field or metric field of the request.

Note: For dimension , this will be set to listing_ids if you are passing listing IDs into the filter parameter.

See the Field Index to learn which calls use key.

localizedName ( string ) [0..?]
This value is translated into the language based on the value of the Accept-Language HTTP request header. For example, Accept-Language is set to de-DE , the value "day" in the dimension container is returned as "tag" and a metric of TRANSACTION is set to "Transaktionsanzahl".
See the Field Index to learn which calls use localizedName.
