ItemLocationImpl

This type is used by the itemLocation container, which provides location information for the item.

Type that uses ItemLocationImpl:

    ItemSummary

Call that uses ItemLocationImpl:

    /item_summary/search GET

Fields

addressLine1 ( string )
addressLine2 ( string )
city ( string )
country ( CountryCodeEnum )
county ( string )
postalCode ( string )
stateOrProvince ( string )


addressLine1 ( string ) [0..?]
The first line of the street address.
See the Field Index to learn which calls use addressLine1.

addressLine2 ( string ) [0..?]
The second line of the street address. This field may contain such values as an apartment or suite number.
See the Field Index to learn which calls use addressLine2.

city ( string ) [0..?]
The city in which the item is located.
See the Field Index to learn which calls use city.

country ( CountryCodeEnum ) [0..?]
The country in which the item is located. These values are two-digit codes defined by the ISO 3166-1 alpha-2 standard, and also shown in the CountryCodeEnum type.
See the Field Index to learn which calls use country.

county ( string ) [0..?]
The county in which the item is located.
See the Field Index to learn which calls use county.

postalCode ( string ) [0..?]
The postal code (or zip code in US) where the item is located.
See the Field Index to learn which calls use postalCode.

stateOrProvince ( string ) [0..?]
The state or province in which the item is located.
See the Field Index to learn which calls use stateOrProvince.
