AvailabilityStatusEnum

Enumeration type that is used to indicate the inventory status of an item.

Types that use AvailabilityStatusEnum:

    Item , ItemDigest

Calls that use one or more values of AvailabilityStatusEnum:

    /item/{item_id} GET , /item_group/{item_group_id} GET

Enumeration Values

Value Description
IN_STOCK Indicates the item variation has stock available for purchase.
LIMITED_STOCK Indicates the item variation has limited stock available for purchase.
OUT_OF_STOCK Indicates the item variation has no stock available for purchase.
