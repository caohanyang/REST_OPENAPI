Amount_0

Contains details about a monetary amount, in the currency used by the call and in the original currency, if the amount was converted between currencies.

Types that use Amount_0:

    Adjustment_0 , GuestPurchaseOrder , LineItem_0 , PricingSummary_0 , PurchaseOrder

Calls that use Amount_0:

    /guest_purchase_order/{purchaseOrderId} GET , /purchase_order/{purchaseOrderId} GET

Fields

currency ( CurrencyCodeEnum ) [0..?]
A three letter ISO 4217 code specifying the currency of the monetary amount.
See the Field Index to learn which calls use currency.

value ( string ) [0..?]
The monetary amount.
See the Field Index to learn which calls use value.
