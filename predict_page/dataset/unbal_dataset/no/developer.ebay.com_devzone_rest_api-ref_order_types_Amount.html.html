Amount

Container defining the monetary value of an amount in the currency used in on the eBay site offering the item and the conversion of that value into another currency.

Types that use Amount:

    Adjustment , LineItem , PricingSummary , Promotion , ShippingOption

Calls that use Amount:

    /checkout_session/initiate POST , /checkout_session/{checkoutSessionId} GET , /checkout_session/{checkoutSessionId}/update_payment_info POST , /checkout_session/{checkoutSessionId}/update_quantity POST , /checkout_session/{checkoutSessionId}/update_shipping_address POST , /checkout_session/{checkoutSessionId}/update_shipping_option POST , /guest_checkout_session/initiate POST , /guest_checkout_session/{checkoutSessionId} GET , /guest_checkout_session/{checkoutSessionId}/update_payment_info POST , /guest_checkout_session/{checkoutSessionId}/update_quantity POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_address POST , /guest_checkout_session/{checkoutSessionId}/update_shipping_option POST

Fields

currency ( CurrencyCodeEnum ) [0..?]
The currency involved in a monetary transaction. Generally, this is the currency used by the country of eBay site offering the item.
See the Field Index to learn which calls use currency.

value ( string ) [0..?]
The dollar value of the currency specified in the currency field. The value of currency defaults to the standard currency used by the country of the eBay site offering the item.
See the Field Index to learn which calls use value.
