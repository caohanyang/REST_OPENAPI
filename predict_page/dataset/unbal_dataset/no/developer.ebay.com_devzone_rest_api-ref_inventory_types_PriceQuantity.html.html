PriceQuantity

This type is used to update the total "ship-to-home" quantity for an inventory item and/or to update the price and/or quantity for a specific offer for that inventory item.

Type that uses PriceQuantity:

    BulkPriceQuantity

Call that uses PriceQuantity:

    /bulk_update_price_quantity POST

Fields

offers ( array of OfferPriceQuantity ) [0..?]
Required if the seller is updating the price and/or quantity of one or more offers. This container is used to update the price and/or quantity for one or more offers associated with the corresponding sku value. It is not required if the seller is only updating the total available quantity for the inventory item through the shipToLocationAvailability container.
See the Field Index to learn which calls use offers.

shipToLocationAvailability ( ShipToLocationAvailability ) [0..?]
Required if the seller is updating the total available quantity for the inventory item. This container is used to specify the total quantity of the inventory item that is available for a standard purchase where the item is shipped to the buyer.
See the Field Index to learn which calls use shipToLocationAvailability.

sku ( string ) [0..?]
This is the seller-defined SKU value of the inventory item that will be updated. This field is required whether the seller is updating the total quantity of the inventory item, or if the seller is updating the price and/or quantity of an active offer associated with the inventory item. Its value must match the SKU value that is associated with the offer(s) specified through the offers container.
See the Field Index to learn which calls use sku.
