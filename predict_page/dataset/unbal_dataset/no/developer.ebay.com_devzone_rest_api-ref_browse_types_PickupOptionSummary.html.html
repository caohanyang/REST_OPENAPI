PickupOptionSummary

This type is used by the pickupOptions container to show the local pickup options that are available for the item.

Type that uses PickupOptionSummary:

    ItemSummary

Call that uses PickupOptionSummary:

    /item_summary/search GET

Field

pickupLocationType ( string ) [0..?]
This container lists the local pickup options available to the buyer. Possible values are ARRANGED_LOCATION and STORE .
See the Field Index to learn which calls use pickupLocationType.
