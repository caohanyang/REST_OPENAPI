Applicable values for /inventory_item/{sku} GETResponse.availability.pickupAtLocationAvailability.fulfillmentTime.unit

BUSINESS_DAY
This enumeration value indicate that business days are used as the time unit for fulfillment time. This means that only business days, or days that the store is open, count towards the fullfillment time, and not days when the store is closed.
CALENDAR_DAY
This enumeration value indicate that calendar days are used as the time unit for fulfillment time. This means that weekends and holidays are also included, and not just business days when the store is open.
DAY
This enumeration value indicate that days are used as the time unit for fulfillment time.
HOUR
This enumeration value indicate that hours are used as the time unit for fulfillment time.
MILLISECOND
This enumeration value indicate that millliseconds are used as the time unit for fulfillment time.
MINUTE
This enumeration value indicate that minutes are used as the time unit for fulfillment time.
MONTH
This enumeration value indicate that months are used as the time unit for fulfillment time.
SECOND
This enumeration value indicate that seconds are used as the time unit for fulfillment time.
YEAR
This enumeration value indicate that years are used as the time unit for fulfillment time.
