Amount

This type is used to express a dollar value.

Types that use Amount:

    Fee , OfferPriceQuantity , PricingSummary , ShippingCostOverride

Calls that use Amount:

    /bulk_update_price_quantity POST , /offer POST , /offer/get_listing_fees POST , /offer/{offer_Id} PUT , /offer/{offerId} GET

Fields

currency ( string ) [0..?]
Required if an amount is being expressed in request payload. A string representation of the type of currency being used.
See the Field Index to learn which calls use currency.

value ( string ) [0..?]
Required if an amount is being expressed in request payload. A string representation of a dollar value expressed in the currency specified in the currency field.
See the Field Index to learn which calls use value.
