OfferPriceQuantity

This type is used by the offers container in a Bulk Update Price and Quantity call to update the current price and/or quantity of a live offer.

Type that uses OfferPriceQuantity:

    PriceQuantity

Call that uses OfferPriceQuantity:

    /bulk_update_price_quantity POST

Fields

availableQuantity ( integer ) [0..?]
Required if the seller is updating the current quantity of the offer. This integer value is used to modify the current quantity of the product (identified with the corresponding sku value) that will be available for purchase in the live offer (identified by the corresponding offerId value).
See the Field Index to learn which calls use availableQuantity.

offerId ( string ) [0..?]
Required if the offers container is used. Unique identifier of the product offer. If an offers container is used to update one or more live offers, the offerId is required in order to identify the active offer to update with a modified price and/or quantity.

If an offerId value is not known, the seller can run a Get Offers call (passing in the correct SKU value as a query parameter) to retrieve offerId values for offers associated with the SKU.
See the Field Index to learn which calls use offerId.

price ( Amount ) [0..?]
Required if the seller is updating the current price of the offer. The dollar value set here will be the new listing price of the product (identified with the corresponding sku value) in the live offer (identified by the corresponding offerId value).
See the Field Index to learn which calls use price.
