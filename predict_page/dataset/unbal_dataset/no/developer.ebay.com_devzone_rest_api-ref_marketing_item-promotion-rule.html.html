Promotion Code Examples

There are 4 types of promotions; BOGO (Buy 1 Get 1), Spend Order, Volume Order, and Order. The table below shows how to use the fields in discountRules.discountBenefit and discountSpecification to create the 13 promotions supported by the Marketing API.

Promotion Name Field Values
BOGO Promotions
Buy 3 Get 2 20% Off discountSpecification. minQuantity = 3
discountSpecification. numberOfDiscountedItems = 2
discountBenefit. percentOffItem = 20

For Each 3 Get 2 20% Off discountSpecification. forEachQuantity = 3
discountSpecification. numberOfDiscountedItems = 2
discountBenefit. percentOffItem = 20

Spend Order Promotions
Spend $60 Get $5 Off discountSpecification. minAmount = 60
discountBenefit. amountOffOrder = 5
For Each $60 Spent Get $5 Off discountSpecification. forEachAmount = 60
discountBenefit. amountOffOrder = 5
Spend at least $50 Get 20% Off Order discountSpecification. minAmount = 50
discountBenefit. amountOffOrder = 20
Volume Order Promotions
Buy 3 Get $10 Off discountSpecification. minQuantity = 3
discountBenefit. amountOffOrder = 10
For Every 3 Get $10 Off discountSpecification. forEachQuantity = 3
discountBenefit. amountOffOrder = 10
Buy 3 Or More Get 20% Off discountSpecification. minQuantity = 3
discountBenefit. percentOffOrder = 20
Order Promotion
$10 Order No Minimum discountSpecification. minAmount = 1
discountBenefit. amountOffOrder = 10
20% Off Order No Minimum discountSpecification. minAmount = 1
discountBenefit. percentOffOrder = 20
$5 Off Each Item discountSpecification. forEachQuantity = 1
discountBenefit. amountOffItem = 5
