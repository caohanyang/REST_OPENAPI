StandardsProfile

A complex type that defines a seller profile.

Type that uses StandardsProfile:

    FindSellerStandardsProfilesResponse

Calls that use StandardsProfile:

    /seller_standards_profile GET , /seller_standards_profile/{program}/{cycle} GET

Fields

cycle ( Cycle )
defaultProgram ( boolean )
evaluationReason ( string )
metrics ( array of Metric )
program ( ProgramEnum )
standardslevel ( StandardsLevelEnum )


cycle ( Cycle ) [0..?]
Indicates the cycle of the evaluation, whether CURRENT or PROJECTED. Each program will have at least one cycle, but a program can include both cycles.
See the Field Index to learn which calls use cycle.

defaultProgram ( boolean ) [0..?]
If set to true , it indicates this is the default program.

Except for sellers in China, a seller's default program is the site where the seller registered with eBay. Seller's in China select their default program when they register.

See the Field Index to learn which calls use defaultProgram.

evaluationReason ( string ) [0..?]
The type of logic used to calculate your overall seller level. eBay may override your calculated seller level if eBay determines there are special circumstances that warrant an override. In general, overrides performed by eBay protect your seller level.
See the Field Index to learn which calls use evaluationReason.

metrics ( array of Metric ) [0..?]
A list of the metrics that contributed to the evaluation. See the applicable metrics and requirements for each program:

eBay Top Rated seller program standards

See the Field Index to learn which calls use metrics.

program ( ProgramEnum ) [0..?]
Indicates the program to which the profile belongs.
See the Field Index to learn which calls use program.

standardslevel ( StandardsLevelEnum ) [0..?]
Indicates the overall standards level of the seller.
See the Field Index to learn which calls use standardslevel.
