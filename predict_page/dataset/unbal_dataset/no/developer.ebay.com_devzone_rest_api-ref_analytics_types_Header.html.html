Header

Type that defines the fields for the dimension key details. The dimension key is the value you set in the request for dimension .

Type that uses Header:

    Report

Call that uses Header:

    /traffic_report GET

Fields

dimensionKeys ( array of Definition ) [0..?]
Container for the fields that describe the dimension key.
See the Field Index to learn which calls use dimensionKeys.

metrics ( array of Definition ) [0..?]
Container for the metric values.
See the Field Index to learn which calls use metrics.
