BulkPriceQuantityResponse

This type is use by the base response payload of the Bulk Update Price and Quantity call. The Bulk Update Price and Quantity call response will return an HTTP status code, offer ID, and SKU value for each offer/inventory item being updated, as well as an errors and/or warnings container if any errors or warnings are triggered while trying to update those offers/inventory items.

Call that uses BulkPriceQuantityResponse:

    /bulk_update_price_quantity POST

Field

responses ( array of PriceQuantityResponse ) [0..?]
This container will return an HTTP status code, offer ID, and SKU value for each offer/inventory item being updated, as well as an errors and/or warnings container if any errors or warnings are triggered while trying to update those offers/inventory items.
See the Field Index to learn which calls use responses.
