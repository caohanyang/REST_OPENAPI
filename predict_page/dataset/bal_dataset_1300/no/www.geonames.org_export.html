GeoNames Data

Data Dump

A daily GeoNames database dump can be downloaded in the form of a large worldwide text file (allCountries.zip). Additional country files are available for convenience. You find the files on our download server . Read the readme.txt for details. A Premium Data Subscription variant is available for professional users.
Postal codes are available as a separate download .
