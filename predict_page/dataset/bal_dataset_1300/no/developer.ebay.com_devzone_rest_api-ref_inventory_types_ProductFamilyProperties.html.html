ProductFamilyProperties

This type is used to specify the details of a motor vehicle that is compatible with the inventory item.

Type that uses ProductFamilyProperties:

    CompatibleProduct

Call that uses ProductFamilyProperties:

    /inventory_item/{sku}/product_compatibility PUT

Fields

engine ( string )
make ( string )
model ( string )
trim ( string )
year ( string )


engine ( string ) [0..?]
The specifications of the engine, including its size, block type, and fuel type. An example is 2.7L V6 gas DOHC naturally aspirated .
