SearchPagedCollection

This type defines the setting that paginate the result sets. The response consists of 0 or more sequenced result sets where each result sets has 0 or more items.

Call that uses SearchPagedCollection (ItemSummary):

    /item_summary/search GET

Fields

href ( string )
itemSummaries ( array of ItemSummary )
limit ( integer )
next ( string )
offset ( integer )
prev ( string )
total ( integer )
warnings ( array of ErrorDetailV3 )


href ( string ) [0..?]
The URI of the request submitted.


For example:



/buy/v1/item/search?q=shirt&price;=[20..80]&limit;=5

This query is for a shirt, that is priced between 20 and 80 dollars and limit the response to 5 items.


See the Field Index to learn which calls use href.

itemSummaries ( array of ItemSummary ) [0..?]
An array of items in one result set. The items are sorted according to the sorting method specified in the request.
See the Field Index to learn which calls use itemSummaries.

limit ( integer ) [0..?]
The maximum number of items that can be returned in a result set. The limit value is set in the request.
See the Field Index to learn which calls use limit.

next ( string ) [0..?]
The URI for the next result set. The following example returns items 11 thru 20 from the list of items.

/buy/v1/item/search?query=t-shirts&limit;=10&offset;=10

See the Field Index to learn which calls use next.

offset ( integer ) [0..?]
This value indicates the current 'page' of items being displayed. This value is 0 for the first page of data, 1 for the second page of data, and so on.
See the Field Index to learn which calls use offset.

prev ( string ) [0..?]
The URI for the previous result set. The following example returns items 1 thru 10 from the list of items.

/buy/v1/item/search?query=t-shirts&limit;=10&offset;=1

See the Field Index to learn which calls use prev.

total ( integer ) [0..?]
The total number of items that match the input criteria.
See the Field Index to learn which calls use total.

warnings ( array of ErrorDetailV3 ) [0..?]
Collection of all warning generated for the input request
See the Field Index to learn which calls use warnings.
