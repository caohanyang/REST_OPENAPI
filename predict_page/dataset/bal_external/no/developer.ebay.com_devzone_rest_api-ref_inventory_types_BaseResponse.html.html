BaseResponse

This is the base response of the createOrReplaceInventoryItem and createOrReplaceProductCompatibility calls.

Calls that use BaseResponse:

    /inventory_item/{sku} PUT , /inventory_item/{sku}/product_compatibility PUT , /inventory_item_group/{inventoryItemGroupKey} PUT

Field

warnings ( array of ErrorDetailV3 ) [0..?]
This container will contain an array of one or more messages if warnings or errors are triggered when a call is made.
See the Field Index to learn which calls use warnings.
