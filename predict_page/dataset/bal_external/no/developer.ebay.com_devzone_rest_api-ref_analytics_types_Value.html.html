Value

Type the defines the value and applicable fields of the metric values.

Type that uses Value:

    Record

Call that uses Value:

    /traffic_report GET

Fields

applicable ( boolean ) [0..?]
This indicates if the value is valid. A value is valid if the service was able to computed it. When this value is false it means one of the values in the calculation was zero and will be rare. For example, buyer is viewing the item at 11:59pm but purchased it at 12:10am, if this was the only view for that day, there would be 1 transaction and no views.

For dimensionValues , this will always be set to true .

See the Field Index to learn which calls use applicable.

value ( any ) [0..?]
The value of the metric or dimension.
See the Field Index to learn which calls use value.
