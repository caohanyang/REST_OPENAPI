OfferKeysWithId

This type is used by the base request payload of the Get Listing Fees call.

Call that uses OfferKeysWithId:

    /offer/get_listing_fees POST

Field

offers ( array of OfferKeyWithId ) [0..?]
This container is used to identify one to 250 unpublished offers for which expected listing fees will be retrieved. The user passes one or more offerId values (maximum of 250) in to this container to identify the offers in which to retrieve expected listing fees. This call is only applicable for offers in the unpublished state.

The call response gives aggregate fee amounts per eBay marketplace, and does not give fee information at the individual offer level.
See the Field Index to learn which calls use offers.
