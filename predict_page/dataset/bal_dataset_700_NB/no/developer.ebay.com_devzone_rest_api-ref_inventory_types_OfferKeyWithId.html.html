OfferKeyWithId

This type is used by the Get Listing Fees call to indicate the unpublished offer(s) for which expected listing fees will be retrieved. The user passes in one or more offerId values (a maximum of 250).

Type that uses OfferKeyWithId:

    OfferKeysWithId

Call that uses OfferKeyWithId:

    /offer/get_listing_fees POST

Field

offerId ( string ) [0..?]
The unique identifier of an unpublished offer for which expected listing fees will be retrieved. One to 250 offerId values can be passed in to the offers container for one Get Listing Fees call. Errors will occur if offerId values representing published offers are passed in.
See the Field Index to learn which calls use offerId.
